Power BI 설치, 로드, 첫 차트

Power BI Desktop을 실행하고 정제된 카페 매출 CSV를 불러와 총매출 카드, 카테고리별 막대그래프, 일자별 꺾은선그래프를 한 페이지에 만든다.

오늘 끝나면 할 수 있는 것

  • Power BI Desktop이 Windows용 보고서 제작 도구라는 점을 알고 설치 또는 실행한다.
  • CSV 파일과 엑셀 파일을 Power BI에 불러오는 경로를 설명한다.
  • 필드 목록, 시각화 패널, 캔버스의 역할을 엑셀 피벗테이블과 비교해 이해한다.
  • category, date, amount 필드를 사용해 막대그래프와 꺾은선그래프를 만든다.
  • amount의 합계를 KPI 카드로 표시하고 표시 단위를 조정해 총매출을 확인한다.
핵심 개념 1

Power BI Desktop은 보고서 제작 도구이다

Power BI Desktop은 Windows PC에 설치해 보고서 파일을 만드는 프로그램이다. 이번 차시는 로컬 파일을 열고 보고서 한 페이지를 만드는 데 집중한다.

  • Power BI Desktop은 Windows에서 보고서를 만드는 프로그램이다.
  • .pbix 파일은 Power BI Desktop에서 저장하는 보고서 파일이다.
  • Power BI Service는 웹에 게시하고 공유하는 서비스이며 S18에서 공유 개념을 다룬다.
  • Mac 수강생은 화면 구조와 개념을 따라오고 S18에서 Looker Studio 경로로 직접 실습한다.

핵심 개념 2

로드는 표를 Power BI 안으로 가져오는 일이다

엑셀은 파일을 열면 셀이 바로 보이지만, Power BI는 데이터를 먼저 가져와야 한다. 이 과정을 로드라고 부른다.

  • CSV는 홈, 데이터 가져오기, 텍스트/CSV 순서로 불러온다.
  • 엑셀 파일은 새 Power BI 파일에서 Excel 통합 문서, cafe_sales_clean.xlsx, sales 시트 순서로 불러온다.
  • 한 Power BI 파일에 CSV와 엑셀을 동시에 불러오지 않는다.
  • 오늘 파일은 정제 실습이 아니므로 date가 날짜로, amount가 숫자로 인식되는지만 확인한다.
핵심 개념 3

시각화는 필드를 놓는 방식으로 만든다

화면 가운데 흰 영역은 캔버스이고, 오른쪽에는 시각화 패널과 데이터 패널이 있다. 차트는 비교 기준, 볼 숫자, 적절한 모양을 정해 만든다.

  • category는 카테고리별 매출 막대그래프의 X축에 놓는다.
  • date는 일자별 매출 꺾은선그래프의 X축에 놓는다.
  • amount는 그래프의 Y축 또는 값 영역에 놓고 합계로 집계한다.
  • 막대그래프는 항목 비교에, 꺾은선그래프는 시간 흐름에, 카드는 핵심 숫자 하나에 쓴다.

첫 페이지에 들어갈 세 시각화

시각화필드 배치목적
총매출 KPI 카드amount 합계핵심 숫자 확인
카테고리별 매출 막대그래프X축 category, Y축 amount 합계상품군별 매출 비교
일자별 매출 꺾은선그래프X축 date, Y축 amount 합계기간 흐름 확인

카테고리별 매출 합계는 커피 45,606,000, 음료 25,550,000, 디저트 19,336,500이다.

핵심 개념 4

KPI 카드는 큰 셀 하나이다

Power BI의 카드는 핵심 숫자 하나를 크게 보여주는 시각화다. 엑셀에서 SUM 결과를 큰 글씨로 적어 두는 것과 같은 역할이다.

  • 이번 미션의 정답 카드 값은 90,492,500이다.
  • 값 필드는 amount이고 집계 방식은 합계여야 한다.
  • 카드가 90.49M 또는 90M처럼 보이면 계산이 틀린 것이 아니라 표시 단위가 자동 축약된 것이다.
  • 정확히 검산하려면 표시 단위를 없음으로, 소수 자릿수를 0으로 바꾼다.

오늘의 미션

미션 1. 총매출 KPI 카드 만들기
카드 또는 카드(새)를 선택하고 amount를 값 영역에 넣은 뒤 합계로 집계해 총매출 90,492,500을 표시한다.
미션 2. 표시 단위 검산하기
카드 서식에서 표시 단위를 없음으로 바꾸고 소수 자릿수를 0으로 맞춰 축약 표시가 아닌 실제 숫자를 확인한다.
미션 3. 첫 페이지 완성하기
같은 페이지에 총매출 카드, 카테고리별 매출 막대그래프, 일자별 매출 꺾은선그래프를 함께 배치하고 S15_cafe_sales_first_dashboard.pbix로 저장한다.

정리

  • Power BI Desktop은 Windows에서 보고서 파일을 만드는 도구이며, 오늘은 공유나 게시가 아니라 로컬 보고서 작성에 집중한다.
  • CSV와 엑셀은 데이터 가져오기 메뉴에서 불러오며, 오늘 실습 결과는 cafe_sales_clean.csv를 기준으로 맞춘다.
  • 필드를 X축, Y축, 값 영역에 놓으면 막대그래프, 꺾은선그래프, 카드가 만들어진다.
  • 총매출 카드의 기준은 amount 합계 90,492,500이며, 표시 단위와 집계 방식을 반드시 확인한다.
다음 차시는 S16 데이터 모델링으로, Power BI에서 여러 표를 한 모델로 연결하고 관계, 스타 스키마, 날짜 차원을 다룬다.
