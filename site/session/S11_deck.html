Pandas 입문: Series, DataFrame, CSV 읽기와 필터

파이썬 문법을 실제 표 데이터로 연결하고, pandas로 CSV를 읽어 구조 확인, 필터, 정렬까지 실습한다.

오늘 끝나면 할 수 있는 것

  • DataFrame을 엑셀 시트, Series를 엑셀의 한 열에 견주어 설명한다.
  • pd.read_csv로 UTF-8 CSV 파일을 읽고, 한글 CSV에서 인코딩 문제가 생길 수 있음을 인지한다.
  • head, info, describe로 데이터의 첫 행, 컬럼, 타입, 기초 통계를 확인한다.
  • 불리언 조건식과 loc로 원하는 행만 필터링하고 sort_values로 금액순 정렬을 한다.
핵심 개념 1

pandas는 표를 다루는 파이썬 도구다

S10에서 배운 변수, 리스트, 딕셔너리, 조건문, 반복문을 표 데이터에 맞게 확장한 도구가 pandas다.

  • import pandas as pd에서 pd는 pandas를 짧게 부르는 별명이다.
  • read_csv, head, describe, sort_values는 엑셀의 열기, 확인, 요약, 정렬에 해당하는 기능이다.
  • 오늘의 목표는 엑셀에서 하던 표 확인, 필터, 정렬을 pandas 코드로 재현하는 것이다.

DataFrame은 표 전체, Series는 열 하나다

엑셀 감각pandas 감각
워크시트 전체DataFrame
열 하나Series
행 번호인덱스
열 제목컬럼명
필터 조건불리언 마스크

처음에는 DataFrame과 Series 이름보다 표 전체와 열 하나라는 감각이 더 중요하다.

핵심 개념 2

CSV 읽기는 경로와 인코딩을 확인하는 일이다

CSV는 쉼표로 구분된 텍스트 파일이므로 파일 경로와 글자 인코딩이 맞아야 열린다. 이번 차시는 정제된 cafe_sales_clean.csv를 사용한다.

  • 파일 경로는 02_datasets/cafe_sales/cafe_sales_clean.csv이다.
  • 실제 확인 결과 데이터 크기는 10,424행 x 12열이고 날짜 범위는 2025-01-01부터 2025-06-30이다.
  • 컬럼은 order_id, date, weekday, hour, store, category, item, unit_price, qty, amount, payment, member이다.
  • UTF-8 CSV는 encoding 옵션 utf-8-sig로 무리 없이 열리며 cp949 원본 처리는 S12에서 다룬다.

읽은 뒤에는 바로 분석하지 않고 먼저 점검한다

확인 도구하는 일엑셀 비교
head앞쪽 몇 행 보기시트 맨 위 훑어보기
info행수, 컬럼, 결측, 타입 보기열 목록과 데이터 형식 확인
describe숫자 열의 기초 통계 보기평균, 최소, 최대 빠른 확인
shape행과 열 개수 보기시트 크기 확인
columns컬럼명 보기머리글 행 확인

이번 데이터의 amount 최소값은 3,000원, 최대값은 19,500원이며 date는 문자열로 읽힌다.

핵심 개념 3

엑셀 필터는 pandas에서 불리언 마스크다

엑셀에서는 지점 필터에서 강남점만 체크하지만, pandas에서는 각 행이 강남점인지 True 또는 False로 먼저 표시한다.

  • 불리언 마스크는 True와 False로 된 Series다.
  • df.loc에 마스크를 넣으면 True인 행만 남고, 강남점 결과는 6,196행 x 12열이다.
  • 조건 여러 개를 연결할 때는 and가 아니라 &를 쓰고 각 조건을 괄호로 감싼다.
핵심 개념 4

sort_values는 금액순 정렬이다

강남점 주문을 만든 뒤 sort_values로 amount를 큰 값부터 본다. 같은 금액은 date, order_id 순서로 보게 했다.

  • ascending=False는 큰 값이 위로 오게 하라는 뜻이다.
  • display_cols는 원본 데이터를 바꾸지 않고 화면에 보여줄 열만 고르는 장치다.
  • sort_values는 원본 df를 자동으로 덮어쓰지 않으므로 결과를 변수에 담아 계속 쓴다.

loc와 iloc는 선택 기준이 다르다

도구기준예시 의미
loc인덱스 라벨과 컬럼명0번 라벨 행의 store 열
iloc숫자 위치0번째 행, 4번째 열

이번 데이터에서 두 방식 모두 첫 행의 지점인 홍대점을 반환한다. 입문 단계에서는 컬럼명을 직접 쓰는 loc가 읽기 쉽다.

정리

  • pandas 입문은 CSV를 읽고, 표의 구조를 확인하고, 조건에 맞는 행만 남기는 흐름에서 시작한다.
  • DataFrame은 표 전체이고 Series는 그 안의 열 하나다.
  • 엑셀 필터는 pandas에서 True 또는 False 마스크를 만들고 loc로 적용하는 과정과 같다.
  • 금액순으로 보려면 sort_values에 정렬할 열과 내림차순 옵션을 준다.
