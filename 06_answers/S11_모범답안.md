## 미션 모범답안

미션:

```text
강남점 주문 중 카테고리가 커피이고, 주문 금액이 10,000원 이상인 행만 추출한다.
그 결과를 금액 큰 순서로 정렬하고, 행수와 총금액과 평균금액을 출력한다.
```

모범 코드:

```python
from pathlib import Path
import pandas as pd

root = Path("/mnt/d/ai-data-analysis-course")
path = root / "02_datasets/cafe_sales/cafe_sales_clean.csv"

df = pd.read_csv(path, encoding="utf-8-sig")

display_cols = ["order_id", "date", "store", "category", "item", "qty", "amount"]

mission_mask = (
    (df["store"] == "강남점")
    & (df["category"] == "커피")
    & (df["amount"] >= 10000)
)

mission = (
    df.loc[mission_mask, display_cols]
    .sort_values(["amount", "date", "order_id"], ascending=[False, True, True])
    .copy()
)

print(f"미션 행수: {len(mission)}")
print(f"미션 총금액: {mission['amount'].sum()}")
print(f"미션 평균금액: {mission['amount'].mean():.2f}")
print(mission.head(10))
```

## 해설

핵심은 조건을 먼저 True/False로 만든 뒤, 그 조건이 True인 행만 남기는 것이다.

```python
mission_mask = (
    (df["store"] == "강남점")
    & (df["category"] == "커피")
    & (df["amount"] >= 10000)
)
```

이 코드는 행마다 세 가지 질문을 한다.

1. 지점이 `강남점`인가
2. 카테고리가 `커피`인가
3. 금액이 10,000원 이상인가

세 조건을 모두 만족해야 하므로 조건 사이에 `&`를 쓴다. pandas에서는 각 조건이 True/False 하나가 아니라 True/False로 된 Series 전체이기 때문에 `and`가 아니라 `&`를 사용한다. 각 조건을 괄호로 감싸는 것도 중요하다.

조건을 적용하는 줄은 아래와 같다.

```python
df.loc[mission_mask, display_cols]
```

앞쪽의 `mission_mask`는 True인 행만 남긴다. 뒤쪽의 `display_cols`는 화면에 보여줄 열만 고른다. 엑셀로 비유하면 필터를 적용한 뒤 필요한 열만 남겨 보는 것이다.

정렬은 아래 줄에서 처리한다.

```python
sort_values(["amount", "date", "order_id"], ascending=[False, True, True])
```

첫 번째 기준인 `amount`는 내림차순이므로 금액이 큰 주문이 위에 온다. 같은 금액이 여러 개 있을 때는 날짜와 주문번호를 오름차순으로 정렬해 결과가 일정하게 보이게 했다.

## 입문자에게 중요한 오류 포인트

`=`와 `==`를 구분한다.

```python
df["store"] == "강남점"
```

위 코드는 같은지 비교하는 조건이다. `df["store"] = "강남점"`처럼 쓰면 필터가 아니라 값을 바꾸는 코드가 된다.

조건 여러 개를 연결할 때는 `and`가 아니라 `&`를 쓴다.

```python
(df["store"] == "강남점") & (df["category"] == "커피")
```

각 조건을 괄호로 감싼다. 괄호를 빼면 연산 순서 때문에 오류가 나거나 의도와 다른 결과가 나올 수 있다.

`df.info()`는 화면에 직접 출력하는 명령이다. `print(df.info())`라고 쓰면 정보 아래에 `None`이 추가로 보일 수 있다. 오류는 아니지만 입문자에게 혼란스럽기 때문에 수업에서는 `df.info()`만 실행한다.
