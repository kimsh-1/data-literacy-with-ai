# S09 모범답안

## 미션 모범답안

상황:

```text
동료가 AI로 cafe_sales_dirty.xlsx를 분석했고,
"총매출은 90,600,500원이며 6월 매출이 가장 높습니다"라고 보고했다.
이 답을 그대로 보고서에 넣어도 되는지 검토해야 한다.
```

모범 판단:

```text
원본을 그대로 합산한 총매출 90,600,500원은 엑셀 피벗 기준값과 일치한다.
그러나 dirty 파일에는 주문번호 중복, 지점명과 카테고리 표기 흔들림, qty=999, amount 음수, 단가×수량과 금액 불일치가 확인된다.
따라서 이 값은 "정제 전 원본 합계"로는 말할 수 있지만, 정제 기준을 정하기 전에는 보고서의 최종 매출로 쓰면 안 된다.
```

## 검증 체크리스트 예시

| 점검 항목 | 기준 또는 확인값 | 판정 |
|---|---:|---|
| 파일명 | `cafe_sales_dirty.xlsx` | 통과 |
| 시트명 | `sales` | 통과 |
| 행수 | 10,439행 | 통과 |
| 사용 금액 열 | `amount` | 통과 |
| 원본 총매출 | 90,600,500원 | 통과 |
| clean 총매출과 차이 | dirty가 108,000원 큼 | 확인 필요 |
| 6월 원본 매출 | 16,822,000원 | 통과 |
| 주문번호 중복 행 | 15행 | 확인 필요 |
| payment 결측 | 313행 | 확인 필요 |
| member 결측 | 209행 | 확인 필요 |
| 날짜 형식 혼재 | `YYYY/MM/DD` 206행, `MM-DD-YYYY` 104행 | 확인 필요 |
| 지점명 표기 흔들림 | `강남 점` 310행 | 확인 필요 |
| category 앞뒤 공백 | 313행 | 확인 필요 |
| 아메리카노 표기 변형 | 106행 | 확인 필요 |
| 수량 이상치 | `qty=999` 2행 | 확인 필요 |
| 음수 금액 | 1행 | 확인 필요 |
| `unit_price × qty = amount` | 불일치 3행 | 확인 필요 |

좋은 답안의 조건:

- AI 답과 피벗 기준값이 일치하는 항목을 구분한다.
- "원본 그대로 합산한 값"과 "최종 확정값"을 구분한다.
- dirty 데이터의 품질 문제를 최소 3가지 이상 언급한다.
- 최종 판단을 `사용 가능`, `조건부 사용`, `사용 보류` 중 하나로 명확히 쓴다.

## AI에게 다시 물을 때의 모범 프롬프트

clean 파일에서 AI 답이 피벗과 다를 때:

```text
내 엑셀 피벗 기준으로 cafe_sales_clean.xlsx의 amount 총합계는 90,492,500원입니다.
방금 답과 차이가 납니다.
amount 열만 합산했는지, 전체 10,424행을 사용했는지, 어떤 필터를 적용했는지 확인해서 다시 계산해 주세요.
```

dirty 파일에서 AI가 품질 문제 없이 결론만 말했을 때:

```text
방금 답은 원본 amount 합계만 말했습니다.
보고서에 쓰기 전에 order_id 중복, store/category 표기 흔들림, qty 이상치, amount 음수, unit_price×qty와 amount 불일치를 점검해 주세요.
각 항목의 건수와 이 문제가 총매출 해석에 주는 영향을 함께 적어 주세요.
```

## 실행한 검증 코드

검증 코드는 아래 파일에 저장되어 있다.

`/mnt/d/ai-data-analysis-course/06_answers/S09_code.py`

실행 명령:

```bash
python3 /mnt/d/ai-data-analysis-course/06_answers/S09_code.py
```

핵심 코드:

```python
from pathlib import Path

import pandas as pd


ROOT = Path("/mnt/d/ai-data-analysis-course")
CLEAN_CSV = ROOT / "02_datasets/cafe_sales/cafe_sales_clean.csv"
DIRTY_CSV = ROOT / "02_datasets/cafe_sales/cafe_sales_dirty.csv"

clean = pd.read_csv(CLEAN_CSV, encoding="utf-8-sig")
dirty = pd.read_csv(DIRTY_CSV, encoding="utf-8-sig")

for df in [clean, dirty]:
    df["amount_num"] = pd.to_numeric(df["amount"], errors="coerce")
    df["qty_num"] = pd.to_numeric(df["qty"], errors="coerce")
    df["unit_price_num"] = pd.to_numeric(
        df["unit_price"].astype(str).str.replace(",", "", regex=False).str.replace("원", "", regex=False),
        errors="coerce",
    )
    df["expected_amount"] = df["unit_price_num"] * df["qty_num"]
    df["date_parsed"] = pd.to_datetime(df["date"], format="mixed", errors="coerce")
    df["month"] = df["date_parsed"].dt.to_period("M").astype(str)
    df["store_norm"] = df["store"].astype(str).str.replace(" ", "", regex=False)
    df["category_norm"] = df["category"].astype(str).str.strip()

clean_total = int(clean["amount_num"].sum())
dirty_total = int(dirty["amount_num"].sum())

clean_store = clean.pivot_table(index="store", values="amount_num", aggfunc="sum")
clean_category = clean.pivot_table(index="category", values="amount_num", aggfunc="sum")
clean_monthly = clean.groupby("month")["amount_num"].sum()

dirty_store_raw = dirty.pivot_table(index="store", values="amount_num", aggfunc="sum")
dirty_category_raw = dirty.pivot_table(index="category", values="amount_num", aggfunc="sum")
dirty_store_norm = dirty.pivot_table(index="store_norm", values="amount_num", aggfunc="sum")
dirty_category_norm = dirty.pivot_table(index="category_norm", values="amount_num", aggfunc="sum")

dirty_issues = {
    "order_id_duplicated_after_first": int(dirty["order_id"].duplicated().sum()),
    "payment_missing": int(dirty["payment"].isna().sum()),
    "member_missing": int(dirty["member"].isna().sum()),
    "store_spaced": int((dirty["store"] == "강남 점").sum()),
    "category_spaced": int((dirty["category"].astype(str) != dirty["category"].astype(str).str.strip()).sum()),
    "qty_999": int((dirty["qty_num"] == 999).sum()),
    "amount_negative": int((dirty["amount_num"] < 0).sum()),
    "amount_mismatch": int((dirty["expected_amount"] != dirty["amount_num"]).sum()),
}

print(clean_total)
print(dirty_total)
print(clean_store)
print(clean_category)
print(clean_monthly)
print(dirty_store_raw)
print(dirty_category_raw)
print(dirty_store_norm)
print(dirty_category_norm)
print(dirty_issues)
```
