# S05 모범답안: 평균 왜곡 사례 찾기

## 미션

`cafe_sales_clean.xlsx`의 `sales` 시트에서 상품별 주문금액 `amount`를 비교한다. 평균이 중앙값보다 많이 높은 상품을 찾아, 평균만 보면 왜 보통 주문금액을 잘못 이해할 수 있는지 설명한다.

## 모범답안

평균 왜곡 사례로 볼 수 있는 상품은 `치즈케이크`다.

| 항목 | 값 |
|---|---:|
| 주문 건수 | 633 |
| 평균 주문금액 | 10,740.92 |
| 중앙값 | 6,500 |
| 최빈값 | 6,500 |
| 최솟값 | 6,500 |
| 최댓값 | 19,500 |
| 평균 - 중앙값 | 4,240.92 |
| 평균 / 중앙값 | 1.65 |

해석:

```text
치즈케이크의 평균 주문금액은 10,740.92원이지만 중앙값과 최빈값은 6,500원이다. 주문 633건 중 331건은 1개 주문이라 6,500원이고, 2개 주문 191건과 3개 주문 111건이 평균을 끌어올린다. 따라서 "보통 치즈케이크 주문은 얼마인가"를 말할 때 평균만 사용하면 실제로 가장 흔한 6,500원 주문을 과장해 볼 수 있다. 매출 계획에는 평균이 유용하지만, 보통 주문금액 설명에는 중앙값과 최빈값을 함께 보는 것이 좋다.
```

## 엑셀 풀이 요약

`item_amount_summary` 시트에서 상품별 요약표를 만든다.

상품 목록:

```excel
=SORT(UNIQUE(sales!$G$2:$G$10425))
```

주문 건수:

```excel
=COUNTIF(sales!$G$2:$G$10425,A2)
```

평균:

```excel
=AVERAGEIF(sales!$G$2:$G$10425,A2,sales!$J$2:$J$10425)
```

중앙값:

```excel
=MEDIAN(FILTER(sales!$J$2:$J$10425,sales!$G$2:$G$10425=A2))
```

최빈값:

```excel
=MODE.SNGL(FILTER(sales!$J$2:$J$10425,sales!$G$2:$G$10425=A2))
```

평균과 중앙값의 차이:

```excel
=C2-D2
```

`평균 - 중앙값` 열을 내림차순으로 정렬하면 `치즈케이크`가 1위다.

## 강사용 검증 코드

아래 코드는 엑셀 풀이와 같은 계산을 pandas로 재현해 검산한 코드다. 이 차시는 엑셀 차시이므로 수강생 실습용 코드는 엑셀 공식이며, 별도 `S05_code.py` 산출물은 만들지 않는다.

```python
from pathlib import Path

import pandas as pd


ROOT = Path("/mnt/d/ai-data-analysis-course")
HR_PATH = ROOT / "02_datasets/hr_employees/hr_employees_clean.csv"
CAFE_PATH = ROOT / "02_datasets/cafe_sales/cafe_sales_clean.csv"

hr = pd.read_csv(HR_PATH)
cafe = pd.read_csv(CAFE_PATH)


def first_mode(values: pd.Series):
    modes = values.mode()
    return modes.iloc[0] if not modes.empty else pd.NA


def outlier_count(values: pd.Series) -> int:
    q1 = values.quantile(0.25)
    q3 = values.quantile(0.75)
    iqr = q3 - q1
    lower = q1 - 1.5 * iqr
    upper = q3 + 1.5 * iqr
    return int(((values < lower) | (values > upper)).sum())


dept_summary = (
    hr.groupby("dept")
    .agg(
        count=("salary", "count"),
        mean=("salary", "mean"),
        median=("salary", "median"),
        mode=("salary", first_mode),
        min=("salary", "min"),
        max=("salary", "max"),
        std_dev=("salary", "std"),
        q1=("salary", lambda s: s.quantile(0.25)),
        q3=("salary", lambda s: s.quantile(0.75)),
    )
    .assign(
        range=lambda d: d["max"] - d["min"],
        iqr=lambda d: d["q3"] - d["q1"],
    )
)
dept_summary["outlier_count"] = hr.groupby("dept")["salary"].apply(outlier_count)
dept_summary = dept_summary.sort_values("mean", ascending=False)

item_summary = (
    cafe.groupby("item")
    .agg(
        count=("order_id", "count"),
        unit_price=("unit_price", "first"),
        mean=("amount", "mean"),
        median=("amount", "median"),
        mode=("amount", first_mode),
        min=("amount", "min"),
        max=("amount", "max"),
        std_dev=("amount", "std"),
    )
    .assign(
        range=lambda d: d["max"] - d["min"],
        mean_minus_median=lambda d: d["mean"] - d["median"],
        mean_to_median_ratio=lambda d: d["mean"] / d["median"],
    )
    .sort_values("mean_minus_median", ascending=False)
)

cheesecake_qty = (
    cafe.loc[cafe["item"] == "치즈케이크"]
    .groupby("qty")
    .agg(count=("order_id", "count"), amount=("amount", "first"))
)

print(f"pandas version: {pd.__version__}")
print(f"HR rows: {len(hr):,}, columns: {hr.shape[1]}")
print(f"Cafe rows: {len(cafe):,}, columns: {cafe.shape[1]}")
print("\nDepartment salary summary")
print(
    dept_summary[
        ["count", "mean", "median", "mode", "min", "max", "range", "std_dev", "outlier_count"]
    ].to_string(float_format=lambda x: f"{x:.2f}")
)
print("\nTop mean distortion candidates")
print(
    item_summary[
        ["count", "mean", "median", "mode", "mean_minus_median", "mean_to_median_ratio"]
    ]
    .head(5)
    .to_string(float_format=lambda x: f"{x:.2f}")
)
print("\nCheesecake quantity distribution")
print(cheesecake_qty.to_string())
```
