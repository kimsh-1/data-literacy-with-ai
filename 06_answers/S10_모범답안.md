# S10 모범답안

## 미션 답안

미션에서 풀 문제입니다. `daily_sales`라는 작은 표에서 월요일부터 금요일까지만 골라내고, 그 행들의 매출 합계와 평균을 구하면 됩니다.

```text
daily_sales에서 평일인 월, 화, 수, 목, 금 행만 골라 weekday_rows에 담고,
평일 매출 합계와 평균을 계산한다.
```

이렇게 풀 수 있습니다.

```python
daily_sales = [
    {"요일": "월", "매출": 120000},
    {"요일": "화", "매출": 98000},
    {"요일": "수", "매출": 135000},
    {"요일": "목", "매출": 170000},
    {"요일": "금", "매출": 155000},
    {"요일": "토", "매출": 210000},
    {"요일": "일", "매출": 190000},
]

weekday_names = ["월", "화", "수", "목", "금"]
weekday_rows = []
weekday_total = 0

for row in daily_sales:
    if row["요일"] in weekday_names:
        weekday_rows.append(row)
        weekday_total = weekday_total + row["매출"]

weekday_average = weekday_total / len(weekday_rows)

print(f"평일 행수: {len(weekday_rows)}")
print(f"평일 매출 합계: {weekday_total}")
print(f"평일 매출 평균: {weekday_average}")
print(f"평일 행: {weekday_rows}")
```

실행하면 평일 행이 5개 골라지고, 그 5개 매출을 더한 값과 평균이 출력됩니다.

```text
평일 행수: 5
평일 매출 합계: 678000
평일 매출 평균: 135600.0
평일 행: [{'요일': '월', '매출': 120000}, {'요일': '화', '매출': 98000}, {'요일': '수', '매출': 135000}, {'요일': '목', '매출': 170000}, {'요일': '금', '매출': 155000}]
```

## 풀이 설명

`daily_sales`는 리스트 안에 딕셔너리가 들어 있는 구조입니다. 어렵게 느껴지면 엑셀 표를 떠올리시면 됩니다. 리스트 전체는 작은 표이고, 딕셔너리 하나는 그 표의 한 행입니다.

```python
{"요일": "월", "매출": 120000}
```

이 한 행에는 이름표가 두 개 붙어 있습니다. `"요일"`이라는 이름표에는 `"월"`이 들어 있고, `"매출"`이라는 이름표에는 `120000`이 들어 있습니다. 그래서 `row["요일"]`이라고 쓰면 지금 보고 있는 행에서 요일 값을 꺼내고, `row["매출"]`이라고 쓰면 매출 값을 꺼냅니다. 사전에서 단어를 찾듯이, 키 이름으로 값을 찾는 방식입니다.

평일인지 확인하는 부분은 이 줄입니다.

```python
if row["요일"] in weekday_names:
```

이 문장은 “지금 보고 있는 행의 요일이 `weekday_names` 안에 있나요?”라고 묻는 문장입니다. `weekday_names`에는 `월`, `화`, `수`, `목`, `금`이 들어 있으므로, 토요일과 일요일은 여기서 걸러집니다. 엑셀 필터에서 요일 항목 중 월요일부터 금요일까지만 체크하는 것과 같은 생각입니다.

조건을 만족한 행에서는 두 가지 일을 합니다.

```python
weekday_rows.append(row)
weekday_total = weekday_total + row["매출"]
```

첫 줄은 평일 행을 `weekday_rows`라는 새 리스트에 모읍니다. 필터링된 행만 따로 담아 두는 것입니다. 둘째 줄은 평일 매출을 계속 더합니다. 키를 재듯이 한 명씩 재고 기록하듯, 반복문은 행을 하나씩 보면서 조건에 맞는 값만 누적합니다.

반복이 끝나면 `weekday_rows`에는 평일 5행만 남습니다. `weekday_total`에는 월요일부터 금요일까지의 매출 합계인 678000이 들어 있습니다.

평균은 합계를 개수로 나누어 계산합니다.

```python
weekday_average = weekday_total / len(weekday_rows)
```

`len(weekday_rows)`는 평일 행이 몇 개인지 세는 함수입니다. 여기서는 5가 됩니다. 그래서 `678000 / 5`를 계산해 평일 매출 평균 `135600.0`이 나옵니다.

## 따라하기 계산 결과

따라하기에서 사용한 매출 리스트입니다. 리스트는 여러 값을 순서대로 담아 둔 상자라고 보면 됩니다.

```python
sales_list = [120000, 98000, 135000, 170000, 155000]
```

반복문으로 하나씩 더하면 아래 결과가 나옵니다.

| 항목 | 값 |
|---|---:|
| 매출 개수 | 5 |
| 매출 합계 | 678000 |
| 매출 평균 | 135600.0 |

반복문 코드는 아래처럼 작성했습니다.

```python
total_sales = 0

for amount in sales_list:
    total_sales = total_sales + amount

average_sales = total_sales / len(sales_list)
```

`total_sales = 0`은 합계를 담을 빈 그릇을 먼저 준비하는 단계입니다. 그다음 `for amount in sales_list:`가 리스트 안의 값을 하나씩 꺼냅니다. 첫 번째에는 `120000`, 두 번째에는 `98000`이 `amount`에 들어갑니다.

`total_sales = total_sales + amount`는 처음 보면 낯설 수 있습니다. 수학의 등식이라기보다 “기존 합계에 이번 값을 더해서 다시 합계에 담는다”라는 뜻입니다. 엑셀의 `SUM`과 `AVERAGE`가 안에서 해 주던 일을 파이썬 문장으로 직접 풀어 쓴 셈입니다.

실제 데이터 분석에서는 `sum()`이나 pandas의 `.sum()`, `.mean()`을 더 자주 씁니다. 그래도 입문 단계에서 반복문으로 직접 더해 보면, 나중에 pandas가 어떤 일을 대신 해 주는지 훨씬 잘 보입니다.

## 함수 맛보기

함수는 자주 쓰는 절차에 이름을 붙여 두는 방법입니다. 평균을 구하는 절차에 `average`라는 이름을 붙이면 아래처럼 쓸 수 있습니다.

```python
def average(numbers):
    total = 0
    for number in numbers:
        total = total + number
    return total / len(numbers)

print(average(sales_list))
```

출력은 아래와 같습니다.

```text
135600.0
```

여기서 `average(numbers)`는 숫자 목록을 받아서 평균을 돌려주는 함수입니다. `return`은 계산한 결과를 함수 밖으로 내보내는 문장입니다.

이번 차시에서는 함수를 깊게 외울 필요는 없습니다. “여러 줄짜리 절차에 이름을 붙여 두고 필요할 때 다시 부르는 것” 정도로 이해하면 충분합니다. 다음 차시부터 볼 `pd.read_csv()`, `df.head()`, `df.mean()`도 정해진 작업에 이름이 붙어 있는 함수 또는 메서드입니다. CSV는 엑셀로도 열리는 표 파일이라고 생각하시면 됩니다. 초보자는 같은 데이터가 엑셀 파일로 제공되면 엑셀 파일을 받아서 시작해도 괜찮습니다.

## 다음 차시와의 연결

오늘의 미션은 행을 하나씩 보면서 조건에 맞는 행만 남겼습니다.

```python
for row in daily_sales:
    if row["요일"] in weekday_names:
        weekday_rows.append(row)
```

여기서 `for`는 행을 하나씩 보는 역할을 했고, `if`는 그 행을 남길지 말지 판단하는 역할을 했습니다. 즉, 오늘 한 일은 “표를 한 줄씩 보면서 원하는 행만 고르는 연습”입니다.

다음 차시 pandas에서는 같은 생각을 표 전체에 한 번에 적용합니다.

```python
df[df["weekday"].isin(["Mon", "Tue", "Wed", "Thu", "Fri"])]
```

이 코드는 `weekday` 열에서 `Mon`, `Tue`, `Wed`, `Thu`, `Fri`에 해당하는 행만 고르는 문장입니다. 오늘 배운 조건문을 표 전체 열에 적용한다고 보면 됩니다. 손으로 한 행씩 확인하던 일을 pandas가 한 줄로 처리해 주는 것이죠.
