# S10 모범답안

## 미션 모범답안

미션:

```text
daily_sales에서 평일인 월, 화, 수, 목, 금 행만 골라 weekday_rows에 담고,
평일 매출 합계와 평균을 계산한다.
```

모범 코드:

```python
daily_sales = [
    {"요일": "월", "매출": 120000},
    {"요일": "화", "매출": 98000},
    {"요일": "수", "매출": 135000},
    {"요일": "목", "매출": 170000},
    {"요일": "금", "매출": 155000},
    {"요일": "토", "매출": 210000},
    {"요일": "일", "매출": 190000},
]

weekday_names = ["월", "화", "수", "목", "금"]
weekday_rows = []
weekday_total = 0

for row in daily_sales:
    if row["요일"] in weekday_names:
        weekday_rows.append(row)
        weekday_total = weekday_total + row["매출"]

weekday_average = weekday_total / len(weekday_rows)

print(f"평일 행수: {len(weekday_rows)}")
print(f"평일 매출 합계: {weekday_total}")
print(f"평일 매출 평균: {weekday_average}")
print(f"평일 행: {weekday_rows}")
```

실행 결과:

```text
평일 행수: 5
평일 매출 합계: 678000
평일 매출 평균: 135600.0
평일 행: [{'요일': '월', '매출': 120000}, {'요일': '화', '매출': 98000}, {'요일': '수', '매출': 135000}, {'요일': '목', '매출': 170000}, {'요일': '금', '매출': 155000}]
```

## 해설

`daily_sales`는 리스트 안에 딕셔너리가 들어 있는 구조다. 리스트 전체는 작은 표이고, 딕셔너리 하나는 표의 한 행이라고 볼 수 있다.

```python
{"요일": "월", "매출": 120000}
```

이 한 행에서 `row["요일"]`은 요일 값을 꺼내고, `row["매출"]`은 매출 값을 꺼낸다.

핵심 조건은 아래 한 줄이다.

```python
if row["요일"] in weekday_names:
```

이 문장은 현재 행의 요일이 `weekday_names` 목록 안에 있으면 아래 코드를 실행하라는 뜻이다. 엑셀에서 요일 필터를 열고 월요일부터 금요일까지 체크하는 상황과 같다.

조건을 만족한 행은 두 가지 처리를 한다.

```python
weekday_rows.append(row)
weekday_total = weekday_total + row["매출"]
```

첫 줄은 평일 행을 별도 리스트에 모은다. 둘째 줄은 평일 매출을 누적한다. 반복이 끝난 뒤 `weekday_rows`에는 평일 5행만 남고, `weekday_total`에는 평일 매출 합계가 남는다.

평균은 합계를 행수로 나누어 계산한다.

```python
weekday_average = weekday_total / len(weekday_rows)
```

## 따라하기 결과

따라하기 매출 리스트:

```python
sales_list = [120000, 98000, 135000, 170000, 155000]
```

반복문으로 계산한 결과:

| 항목 | 값 |
|---|---:|
| 매출 개수 | 5 |
| 매출 합계 | 678000 |
| 매출 평균 | 135600.0 |

반복문 코드:

```python
total_sales = 0

for amount in sales_list:
    total_sales = total_sales + amount

average_sales = total_sales / len(sales_list)
```

이 코드는 엑셀의 `SUM`과 `AVERAGE`를 내부 절차로 풀어 쓴 것이다. 실제 분석에서는 `sum()`이나 pandas의 `.sum()`, `.mean()`을 더 자주 쓰지만, 입문 단계에서는 반복문으로 직접 더해 보는 것이 도움이 된다.

## 함수 맛보기

함수로 평균 계산 절차에 이름을 붙이면 아래와 같다.

```python
def average(numbers):
    total = 0
    for number in numbers:
        total = total + number
    return total / len(numbers)

print(average(sales_list))
```

출력:

```text
135600.0
```

이번 차시에서 함수는 개념만 이해하면 된다. 다음 차시부터 볼 `pd.read_csv()`, `df.head()`, `df.mean()`도 정해진 작업에 이름이 붙은 함수 또는 메서드라고 생각하면 된다.

## 다음 차시 연결

오늘의 미션은 행을 하나씩 보면서 조건에 맞는 행만 남겼다.

```python
for row in daily_sales:
    if row["요일"] in weekday_names:
        weekday_rows.append(row)
```

다음 차시 pandas에서는 같은 생각을 표 전체에 한 번에 적용한다.

```python
df[df["weekday"].isin(["Mon", "Tue", "Wed", "Thu", "Fri"])]
```

즉, pandas 필터는 오늘 배운 조건문을 열 전체에 적용하는 방법이다.
