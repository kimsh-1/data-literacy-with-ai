# S17 모범답안

## 미션 요약

미션은 `shop_orders` 데이터로 시계열 KPI 대시보드 한 페이지를 만드는 것이다. 완성 대시보드는 월별 총매출 흐름, 고객 등급 슬라이서, 카테고리 드릴다운, 전월대비 성장률을 함께 보여주어야 한다.

Power BI 파일 자체를 자동 생성하지는 않는다. 대신 아래 DAX 측정값과 검증값을 기준으로 수강생 화면을 확인한다.

## 데이터 검증 결과

pandas 3.0.3으로 실제 파일을 열어 확인했다.

| 파일 | 실제 행수 | 열수 |
|---|---:|---:|

추가 확인:

| 항목 | 값 |
|---|---:|
| 주문 기간 | 2024-01-01 ~ 2024-12-31 |
| 주문 월수 | 12개월 |
| 날짜테이블 행수 | 366 |
| 고객 조인 누락 | 0건 |
| 상품 조인 누락 | 0건 |
| 전체 주문건수 | 20,000 |

| 전체 매출 | 1,938,489,803 |
| 평균주문금액 | 96,924.49 |

매출 계산식:

```text
매출 = products[price] × orders[qty] × (1 - orders[discount_rate])
```

## 모델 관계 정답

Power BI 모델 보기에서 아래 관계가 있어야 한다.

| 1쪽 | 다쪽 | 방향 |
|---|---|---|
| `customers[customer_id]` | `orders[customer_id]` | 단일 |
| `products[product_id]` | `orders[product_id]` | 단일 |
| `날짜테이블[Date]` | `orders[order_date]` | 단일 |

`날짜테이블`은 날짜 테이블로 표시하고, 날짜 열은 `Date`로 지정한다.

## 날짜테이블 DAX

S16 날짜테이블이 없을 때는 아래 DAX로 만든다.

```DAX
날짜테이블 =
ADDCOLUMNS(
    CALENDAR(DATE(2024, 1, 1), DATE(2024, 12, 31)),
    "연도", YEAR([Date]),
    "월번호", MONTH([Date]),
    "연월", FORMAT([Date], "yyyy-MM"),
    "연월정렬", YEAR([Date]) * 100 + MONTH([Date])
)
```

생성 후 `날짜테이블[연월]`은 `날짜테이블[연월정렬]` 기준으로 정렬한다.

## 측정값 모범답안

### 총매출

```DAX
총매출 =
SUMX(
    orders,
    RELATED(products[price]) * orders[qty] * (1 - orders[discount_rate])
)
```

검증값:

```text
1,938,489,803
```

### 주문건수

```DAX
주문건수 =
DISTINCTCOUNT(orders[order_id])
```

검증값:

```text
20,000
```

### 총수량

```DAX
총수량 =
SUM(orders[qty])
```

검증값:

```text
29,535
```

### 평균주문금액

```DAX
평균주문금액 =
DIVIDE([총매출], [주문건수])
```

검증값:

```text
96,924.49
```

### 매출비중

```DAX
매출비중 =
DIVIDE(
    [총매출],
    CALCULATE([총매출], ALL(products[category]))
)
```

카테고리별 주요 검증값:

| 카테고리 | 매출 | 매출비중 |
|---|---:|---:|
| 디지털 | 673,862,749 | 34.76% |
| 패션 | 281,835,909 | 14.54% |
| 스포츠 | 217,531,145 | 11.22% |
| 뷰티 | 179,845,362 | 9.28% |
| 생활용품 | 178,042,374 | 9.18% |

### 전월매출

```DAX
전월매출 =
CALCULATE(
    [총매출],
    SAMEPERIODLASTMONTH('날짜테이블'[Date])
)
```

### 전월대비 증감액

```DAX
전월대비증감액 =
[총매출] - [전월매출]
```

### 전월대비 성장률

```DAX
전월대비성장률 =
DIVIDE([전월대비증감액], [전월매출])
```

## 미션 대시보드 구성 정답

권장 배치:

1. 상단: 총매출, 주문건수, 평균주문금액, 전월대비 성장률 카드
2. 왼쪽: 고객 등급 슬라이서, 월 슬라이서
3. 가운데: 월별 총매출 선그래프
4. 오른쪽: 카테고리별 매출 막대그래프
5. 하단: 카테고리별 매출비중 표와 해석 문장

필수 동작:

- 월 슬라이서에서 `2024-12`를 선택하면 전월대비 성장률 카드가 1.14%를 표시한다.
- 고객 등급 슬라이서에서 `VIP`를 선택하면 총매출 카드가 393,432,930을 표시한다.
- 카테고리 막대그래프에서 `디지털`을 클릭하면 다른 시각화가 디지털 기준으로 필터링된다.
- 카테고리 막대그래프는 `products[category]`에서 `products[product_name]`으로 드릴다운된다.
