## 미션 질문에 대한 답

미션 질문은 "매출액 합계 기준으로 가장 잘 팔리는 시간대는 언제인가?"입니다.

이 질문에서 기준은 분명히 매출액 합계입니다. 판매수량이 많은 시간이나 주문건수가 많은 시간이 아니라, 각 시간대의 주문금액을 모두 더했을 때 가장 큰 시간을 찾는 문제입니다. 키를 잴 때 자를 하나 정하듯이, 여기서는 `amount`의 합계를 기준자로 삼는다고 보면 됩니다.

정답:

매출액 합계 기준 가장 잘 팔리는 시간대는 13시이며, 매출은 **11,626,000원**, 판매수량은 2,220개, 주문건수는 1,348건이다.

이 데이터에서는 판매수량 기준과 주문건수 기준으로 봐도 13시가 1위입니다. 그래서 "매출은 13시가 큰데 주문은 다른 시간대가 많았다"는 상황이 아니라, 매출액·판매수량·주문건수가 모두 13시에 가장 높게 모인 결과라고 해석할 수 있습니다.

## 시간대별 매출 상위 결과 읽기

| 순위 | 시간대 | 주문건수 | 판매수량 | 매출액 |
|---:|---:|---:|---:|---:|
| 1 | 13시 | 1,348 | 2,220 | 11,626,000 |
| 2 | 12시 | 1,179 | 1,937 | 10,205,000 |
| 3 | 14시 | 1,079 | 1,778 | 9,381,500 |
| 4 | 15시 | 815 | 1,362 | 7,119,500 |
| 5 | 16시 | 829 | 1,348 | 7,085,500 |

해석:

여기서 뭐가 보이시나요? 점심 직후인 13시가 매출액, 판매수량, 주문건수 모두에서 가장 큽니다. 12시와 14시도 높은 편이지만, 매출액 합계로 비교하면 13시가 가장 앞섭니다.

따라서 이 데이터만 놓고 보면 13시를 가장 잘 팔리는 시간대로 판단할 수 있습니다. "잘 팔린다"는 말을 감으로 말하지 않고, 매출액 합계라는 기준으로 확인했다는 점이 중요합니다.

## 따라하기 1 검증 결과: 지점별·카테고리별 매출

| 지점 | 디저트 | 음료 | 커피 | 총합계 |
|---|---:|---:|---:|---:|
| 강남점 | 11,604,000 | 15,281,500 | 26,539,500 | 53,425,000 |
| 홍대점 | 7,732,500 | 10,268,500 | 19,066,500 | 37,067,500 |
| 총합계 | 19,336,500 | 25,550,000 | 45,606,000 | 90,492,500 |

해석:

강남점과 홍대점 모두 커피 매출이 가장 큽니다. 표를 가로로 읽어 보면 각 지점 안에서 어떤 카테고리가 큰지 볼 수 있고, 세로로 읽어 보면 같은 카테고리에서 어느 지점이 더 큰지 볼 수 있습니다.

전체 매출은 강남점이 53,425,000원으로 홍대점보다 큽니다. 즉, 이 표에서는 "강남점이 전체 매출을 더 많이 만들고 있고, 두 지점 모두 커피가 가장 큰 매출 카테고리다"라고 말할 수 있습니다.

## 따라하기 2 검증 결과: 요일별·시간대별 매출

요일×시간대를 함께 보았을 때 매출액 합계가 가장 큰 조합은 `Sun` 13시이며, 매출액은 2,260,500원입니다.

`Sun`은 일요일을 뜻합니다. 그러니까 "전체 시간대만 보면 13시가 가장 크고, 요일까지 함께 나누어 보면 일요일 13시가 가장 큰 칸이다"라고 이해하면 됩니다. 사전에서 단어를 찾듯이, `Sun`이라는 요일과 `13시`라는 시간대를 교차해서 해당 칸의 금액을 읽은 것입니다.

## 검증용 pandas 코드

아래 코드는 엑셀 피벗테이블에서 한 계산을 pandas로 다시 확인한 것입니다. pandas는 파이썬에서 표 데이터를 다룰 때 많이 쓰는 도구입니다. 수업에서는 엑셀 피벗테이블로 답을 찾지만, 이렇게 코드로도 같은 결과가 나오는지 검산할 수 있습니다.

학생 입장에서는 이 코드를 외울 필요는 없습니다. 여기서는 "엑셀에서 만든 피벗 결과가 실제 원본 데이터와 맞는지 한 번 더 확인했다"는 뜻으로 보면 됩니다.

```python
from pathlib import Path

import pandas as pd

data_path = Path("/mnt/d/ai-data-analysis-course/02_datasets/cafe_sales/cafe_sales_clean.csv")
df = pd.read_csv(data_path)

assert df.shape == (10424, 12)
assert df["amount"].isna().sum() == 0
assert df["qty"].isna().sum() == 0

store_category = pd.pivot_table(
    df,
    index="store",
    columns="category",
    values="amount",
    aggfunc="sum",
    margins=True,
    margins_name="총합계",
).fillna(0).astype(int)

hour_summary = (
    df.groupby("hour")
    .agg(
        주문건수=("order_id", "count"),
        판매수량=("qty", "sum"),
        매출액=("amount", "sum"),
    )
    .reset_index()
    .sort_values("매출액", ascending=False)
)

weekday_order = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
weekday_hour = (
    pd.pivot_table(
        df,
        index="weekday",
        columns="hour",
        values="amount",
        aggfunc="sum",
    )
    .reindex(weekday_order)
    .fillna(0)
    .astype(int)
)

top_hour = hour_summary.iloc[0]
top_weekday_hour_raw = weekday_hour.stack().sort_values(ascending=False).index[0]
top_weekday_hour = (top_weekday_hour_raw[0], int(top_weekday_hour_raw[1]))
top_weekday_hour_amount = int(weekday_hour.stack().sort_values(ascending=False).iloc[0])

print("데이터 행열:", df.shape)
print("전체 매출:", int(df["amount"].sum()))
print("전체 판매수량:", int(df["qty"].sum()))
print()
print("[지점 x 카테고리 매출]")
print(store_category)
print()
print("[시간대별 상위 5개]")
print(hour_summary.head(5).to_string(index=False))
print()
print(
    "매출 1위 시간대:",
    f"{int(top_hour['hour'])}시",
    f"매출 {int(top_hour['매출액']):,}원",
    f"판매수량 {int(top_hour['판매수량']):,}개",
    f"주문건수 {int(top_hour['주문건수']):,}건",
)
print(
    "요일 x 시간대 최고 조합:",
    top_weekday_hour,
    f"{top_weekday_hour_amount:,}원",
)
```
