# S14 모범답안

## 미션 목표

`cafe_sales_clean.csv`를 사용해 월별 매출 추세 라인차트와 BI 핸드오프용 CSV를 저장한다.

조건:

- `date`를 날짜형으로 변환한다.
- 월별 `amount` 합계를 계산한다.
- 월별 주문수와 객단가도 함께 계산한다.
- 월별 매출 라인차트를 PNG로 저장한다.
- 월별 집계표를 CSV로 저장한다.
- 6월 매출이 가장 높고, 2월 매출이 가장 낮은지 확인한다.

## 미션 모범 코드

아래 코드는 미션만 독립적으로 실행할 수 있는 최소 코드다. 전체 따라하기와 검증 코드는 `/mnt/d/ai-data-analysis-course/06_answers/S14_code.py`에 저장되어 있다.

```python
from pathlib import Path

import matplotlib.pyplot as plt
import matplotlib.ticker as mticker
import pandas as pd
import seaborn as sns
from matplotlib import font_manager

ROOT = Path("/mnt/d/ai-data-analysis-course")
DATA = ROOT / "02_datasets"
OUT = ROOT / "06_answers/S14_outputs"
OUT.mkdir(parents=True, exist_ok=True)

font_candidates = [
    "NanumGothic",
    "NanumGothicCoding",
    "Noto Sans CJK KR",
    "Malgun Gothic",
    "AppleGothic",
    "DejaVu Sans",
]
available_fonts = {font.name for font in font_manager.fontManager.ttflist}
selected_font = next(
    (font for font in font_candidates if font in available_fonts),
    "DejaVu Sans",
)

sns.set_theme(style="whitegrid")
plt.rcParams["font.family"] = selected_font
plt.rcParams["axes.unicode_minus"] = False

cafe = pd.read_csv(DATA / "cafe_sales/cafe_sales_clean.csv")
cafe["date"] = pd.to_datetime(cafe["date"], errors="coerce")
cafe = cafe.dropna(subset=["date", "amount"]).copy()

monthly_sales = (
    cafe.groupby(pd.Grouper(key="date", freq="MS"))
    .agg(매출합계=("amount", "sum"), 주문수=("order_id", "count"))
    .reset_index()
    .rename(columns={"date": "월"})
)
monthly_sales["객단가"] = monthly_sales["매출합계"] / monthly_sales["주문수"]

monthly_sales.to_csv(
    OUT / "s14_monthly_sales.csv",
    index=False,
    encoding="utf-8-sig",
)

fig, ax = plt.subplots(figsize=(8.8, 5.2))
sns.lineplot(
    data=monthly_sales,
    x="월",
    y="매출합계",
    marker="o",
    color="#A3D576",
    linewidth=1.2,
    ax=ax,
)

ax.set_title("카페 월별 매출 추세")
ax.set_xlabel("월")
ax.set_ylabel("매출합계")
ax.yaxis.set_major_formatter(
    mticker.FuncFormatter(lambda x, _: f"{x / 1_000_000:.1f}백만")
)

plt.tight_layout()
plt.savefig(OUT / "s14_monthly_sales_trend.png", dpi=160)
plt.close(fig)

print("선택된 글꼴:", selected_font)
print(monthly_sales)
print("CSV:", OUT / "s14_monthly_sales.csv")
print("차트:", OUT / "s14_monthly_sales_trend.png")
```

## 주요 전처리와 저장 파일

### 1. 서울 지하철 시간대 히트맵

사용 파일:

```text
02_datasets/real_korean/seoul_subway/seoulmetro_daily_hourly_2025.csv
```

전처리:

- `승하차구분 == "승차"` 행만 사용한다.
- `호선 + 역명`으로 `역라벨`을 만든다.
- 20개 시간대 열을 역별로 합산한다.
- 총 승차 상위 10개 역만 히트맵에 사용한다.
- 10개 역 × 20개 시간대를 긴 표로 바꾸어 200행 CSV로 저장한다.

저장 파일:

### 2. 강남 아파트 실거래 차트

사용 파일:

```text
02_datasets/real_korean/realestate_apt/molit_apt_sale_seoul_gangnam_2024.csv
```

전처리:

- `거래금액(만원)`의 쉼표를 제거하고 숫자형 `거래금액_만원`을 만든다.
- `거래금액_억원 = 거래금액_만원 / 10,000`을 만든다.
- `계약년월`을 날짜형 `계약월`로 변환한다.
- 전용면적을 5개 `면적구간`으로 나눈다.
