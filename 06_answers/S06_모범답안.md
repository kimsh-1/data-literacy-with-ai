# S06 모범답안

## 미션 모범답안

주장 문장은 아래처럼 한 줄입니다. 이 문장은 아직 결론이 아니라, 데이터로 확인해야 할 가설입니다. 가설은 “그럴듯하다”에서 멈추면 안 되고, 실제 숫자로 확인해야 합니다.

```text
기온이 오르면 매출이 는다.
```

모범 반박은 아래처럼 씁니다. 여기서 중요한 점은 기온과 매출을 같은 날짜 기준으로 맞춘 뒤, 두 값이 정말 같이 움직이는지 먼저 확인하는 것입니다. 키와 몸무게를 비교할 때 한 사람씩 짝을 맞춰 보듯이, 날짜도 서로 맞춰야 비교가 됩니다.

```text
이 주장을 확인하려면 같은 날짜 기준의 기온 데이터와 매출 데이터가 필요하다. 먼저 날짜별 기온과 매출의 산점도를 그리고 상관계수를 계산해 두 값이 실제로 함께 움직이는지 확인해야 한다. 상관계수가 0에 가깝다면 이 주장은 데이터로 지지되지 않는다. 상관계수가 높게 나오더라도 기온이 매출을 직접 올렸다고 바로 말할 수는 없다. 계절, 요일, 휴일, 행사, 광고, 가격 변화, 매장 수 변화 같은 다른 요인이 매출에 영향을 주었을 수 있으므로 비슷한 조건끼리 비교하거나 추가 변수를 통제해야 한다.
```

좋은 답안은 단순히 “맞다” 또는 “아니다”라고 끝내지 않습니다. 어떤 데이터를 봐야 하는지, 어떤 방법으로 확인해야 하는지, 그리고 왜 바로 원인이라고 말하면 안 되는지를 함께 설명해야 합니다.

좋은 답안의 조건:

- "같은 날짜 기준의 기온과 매출"처럼 비교 단위를 맞추어야 한다고 쓴다.
- 산점도와 상관계수로 먼저 확인해야 한다고 쓴다.
- 상관이 있어도 인과는 아니라고 쓴다.
- 요일, 계절, 행사, 광고, 가격, 매장 수 같은 다른 요인을 2개 이상 언급한다.

## 따라하기 결과 해설

### 1. 아파트 전용면적과 거래금액 관계 읽기

검증에 사용한 데이터는 수업 홈의 데이터 받기 페이지에서 내려받은 강남구 아파트 실거래 파일입니다. 엑셀로도 열리는 표 파일인 CSV를 받아도 되고, 초보자는 엑셀 파일(.xlsx)이 제공되어 있다면 그 파일을 받아서 열어도 괜찮습니다.

실제 확인 결과:

- 행수: 3,754행
- 사용 열: `전용면적(㎡)`, `거래금액(만원)`
- 전처리: `거래금액(만원)`의 쉼표를 제거하고 숫자형 `거래금액_만원`으로 변환
- Pearson 상관계수: `r = 0.806`

여기서 행수 3,754행은 아파트 거래가 3,754건 들어 있다는 뜻입니다. `전용면적(㎡)`은 집의 크기이고, `거래금액(만원)`은 실제로 거래된 가격입니다. 가격에 쉼표가 들어 있으면 엑셀이 글자처럼 읽을 수 있으므로, 계산을 하려면 먼저 숫자로 바꿔야 합니다.

Pearson 상관계수는 두 숫자 열이 얼마나 함께 움직이는지 보는 값입니다. `r = 0.806`은 1에 가까운 양수이므로, 전용면적이 커질수록 거래금액도 대체로 커지는 흐름이 강하게 보인다는 뜻입니다. 다만 이것은 “면적 하나만 가격을 결정한다”는 말은 아닙니다.

해석:

```text
강남구 아파트 실거래 3,754건에서 전용면적과 거래금액은 강한 양의 상관을 보인다. 전용면적이 클수록 거래금액도 대체로 높다. 그러나 같은 면적이라도 단지, 위치, 층, 연식, 거래 시점이 다르므로 전용면적만이 가격의 원인이라고 단정할 수는 없다.
```

### 2. HR 데이터로 심슨의 역설 확인하기

검증에 사용한 데이터는 수업 홈의 데이터 받기 페이지에서 내려받은 HR 직원 데이터 파일입니다. HR 데이터는 직원의 부서, 성별, 연봉 같은 정보가 들어 있는 연습용 데이터라고 보면 됩니다.

실제 확인 결과:

- 행수: 1,200행
- 전체 평균 연봉
  - 남성: 67,296,571원
  - 여성: 63,910,600원
  - 남성 - 여성: 3,385,971원
- 부서별 평균: 5개 부서 모두 여성 평균이 남성 평균보다 높음

전체 평균만 보면 남성 평균 연봉이 67,296,571원이고, 여성 평균 연봉이 63,910,600원입니다. 그래서 성별만 놓고 보면 남성이 3,385,971원 높아 보입니다.

그런데 여기서 멈추면 위험합니다. 직원들은 모두 같은 일을 하는 것이 아니라 서로 다른 부서에 속해 있습니다. 연봉은 성별뿐 아니라 부서의 영향을 크게 받을 수 있죠. 그래서 사전에서 단어를 더 정확히 찾으려면 한 글자만 보지 않고 앞뒤 문맥을 보듯이, 평균도 부서라는 문맥을 함께 봐야 합니다.

부서별 평균 연봉:

| 부서 | 남성 평균 | 여성 평균 | 여성 - 남성 |
|---|---:|---:|---:|
| 개발본부 | 73,541,000원 | 76,974,286원 | 3,433,286원 |
| 고객운영팀 | 51,497,500원 | 54,095,000원 | 2,597,500원 |
| 데이터전략팀 | 67,437,778원 | 72,667,143원 | 5,229,365원 |
| 영업본부 | 61,020,000원 | 65,530,909원 | 4,510,909원 |
| 인사총무팀 | 58,280,000원 | 59,850,769원 | 1,570,769원 |

표를 보면 개발본부, 고객운영팀, 데이터전략팀, 영업본부, 인사총무팀 모두에서 여성 평균이 남성 평균보다 높습니다. 전체로 봤을 때의 결론과 부서별로 나눠 봤을 때의 결론이 뒤집힌 셈입니다. 이런 현상을 심슨의 역설이라고 합니다.

해석:

```text
전체 평균만 보면 남성 평균 연봉이 여성보다 높다. 그러나 부서별로 나누면 5개 부서 모두에서 여성 평균 연봉이 남성보다 높다. 이는 부서 구성의 차이가 전체 평균에 영향을 준 심슨의 역설 사례다. 따라서 성별 전체 평균만으로 결론을 내리면 중요한 그룹 차이를 놓칠 수 있다.
```

### 3. 참고 반례: 배달 통화건수와 기온

참고로 확인한 데이터는 수업 홈의 데이터 받기 페이지에서 내려받은 배달 통화건수와 기온 파일입니다. 이 데이터는 상관 분석의 주자료라기보다, “그럴듯한 말도 실제로 확인해 봐야 한다”는 점을 보여 주는 예시로 봅니다.

실제 확인 결과:

- 행수: 1,947행
- 사용 열: `평균기온(°C)`, `통화건수`
- Pearson 상관계수: `r = 0.015`

기온과 배달은 얼핏 관련 있어 보입니다. 예를 들어 추운 날에는 밖에 나가기 싫어서 배달을 더 시킬 것 같다고 생각할 수 있죠. 하지만 실제 데이터에서 `r = 0.015`라면 거의 0에 가깝습니다. 두 값이 함께 움직이는 흐름이 거의 보이지 않는다는 뜻입니다.

해석:

```text
배달 통화건수와 평균기온은 직관적으로 관련 있어 보일 수 있지만, 이 참고 데이터에서는 상관계수가 거의 0이다. 그러므로 "추우면 배달이 늘어난다" 같은 말은 실제 데이터 확인 전에는 결론으로 말할 수 없다.
```

이 참고 반례는 수업에서 상관 분석의 주자료로 쓰지 않고, 그럴듯한 주장도 확인이 필요하다는 설명에만 사용합니다.

## 실행한 검증 코드

아래 코드는 위 결과가 실제 데이터에서 나온 값인지 확인하기 위해 사용한 검증용 파이썬 코드입니다. 학생이 엑셀로 따라 할 때는 같은 의미의 작업을 엑셀 함수와 피벗테이블로 하는 것입니다.

실행 명령:

```bash
python3 06_answers/S06_code.py
```

핵심 코드:

```python
from pathlib import Path

import pandas as pd


ROOT = Path(__file__).resolve().parents[1]
APT_PATH = ROOT / "02_datasets/real_korean/realestate_apt/molit_apt_sale_seoul_gangnam_2024.csv"
HR_PATH = ROOT / "02_datasets/hr_employees/hr_employees_clean.csv"
DELIVERY_PATH = ROOT / "02_datasets/real_korean/delivery_food/food_delivery_prediction_raw_data2.csv"

apt = pd.read_csv(APT_PATH)
apt["거래금액_만원"] = pd.to_numeric(
    apt["거래금액(만원)"].astype(str).str.replace(",", "", regex=False),
    errors="coerce",
)
apt_corr = apt["전용면적(㎡)"].corr(apt["거래금액_만원"])

hr = pd.read_csv(HR_PATH)
hr_overall = hr.groupby("gender")["salary"].mean().reindex(["남성", "여성"])
hr_by_dept = hr.pivot_table(index="dept", columns="gender", values="salary", aggfunc="mean")
hr_by_dept["여성-남성"] = hr_by_dept["여성"] - hr_by_dept["남성"]

delivery = pd.read_csv(DELIVERY_PATH)
delivery_corr = delivery["평균기온(°C)"].corr(delivery["통화건수"])

print(f"아파트 행수: {len(apt):,}")
print(f"전용면적-거래금액 상관계수: {apt_corr:.3f}")
print(f"HR 행수: {len(hr):,}")
print(hr_overall)
print(hr_by_dept)
print(f"배달 참고 데이터 행수: {len(delivery):,}")
print(f"평균기온-통화건수 상관계수: {delivery_corr:.3f}")
```
