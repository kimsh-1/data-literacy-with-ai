# S20 모범답안

## 미션 모범답안

S19에서 상권 데이터를 구별·업종별 점포수로 집계한 수강생의 예시 답안이다.

### 1페이지 보고서 예시

제목:

```text
서울 음식점·카페 상권은 강남구에 가장 집중되어 있으며, 구별 우선순위는 한식과 커피점/카페 조합부터 확인해야 한다.
```

핵심 발견:

```text
분석 대상은 서울 음식점·카페 상권 데이터 119,158행이다.
구별 점포수는 강남구 12,095개, 마포구 7,396개, 송파구 7,106개 순으로 많다.
구별·업종 조합은 강남구 한식 3,917개, 마포구 한식 2,835개, 송파구 한식 2,626개, 강남구 커피점/카페 2,253개가 상위에 온다.
```

핵심 시각화:

```text
구별·업종 조합 상위 10개 막대그래프를 사용한다.
이 차트는 어느 구와 업종 조합에 점포가 집중되어 있는지, 다음 조사 우선순위를 어디에 둘지 직접 보여준다.
```

시사점:

```text
상권 후보를 고를 때는 전체 점포수가 많은 구를 먼저 보되, 실제 실행 우선순위는 업종별 밀집도까지 함께 보아야 한다. 강남구는 전체 점포수와 한식·커피점/카페 점포수가 모두 높으므로 1차 조사 후보로 적합하다.
```

권장 액션:

```text
1. 상권 조사 담당자는 강남구, 마포구, 송파구를 1차 비교 대상으로 잡는다.
2. 각 구는 한식과 커피점/카페 점포수를 먼저 비교하고, 경쟁 강도가 높은 조합을 표시한다.
3. 회의 자료에는 구별 점포수 막대그래프 1개와 구별·업종 상위 10개 표 1개만 남기고, 원자료 전체 표는 부록으로 분리한다.
```

주의점:

```text
이 분석은 2021년 서울 음식점·카페 상권 데이터의 점포수만 사용한 교육용 예시다. 실제 출점 판단에는 임대료, 유동인구, 매출, 배달 수요, 폐업률, 접근성 같은 추가 정보가 필요하다.
```

동료 피드백 반영 예시:

```text
받은 피드백 1: 제목은 결론이 보이지만 "무엇을 해야 하는지"가 조금 약하다.
반영 내용 1: 액션 영역에 "강남구, 마포구, 송파구를 1차 비교 대상으로 잡고 한식·커피점/카페 점포수를 먼저 비교한다"를 명확히 추가했다.

받은 피드백 2: 강남구가 왜 먼저인지 근거가 부족하다.
반영 내용 2: 구별 점포수 상위 3개와 구별·업종 조합 상위 값을 발견 문장에 추가했다.
```

### 3분 발표 원고 예시

```text
서울 음식점·카페 상권은 강남구에 가장 집중되어 있으며, 구별 우선순위는 한식과 커피점/카페 조합부터 확인해야 합니다.

분석 대상은 119,158행입니다. 구별 점포수는 강남구 12,095개, 마포구 7,396개, 송파구 7,106개 순으로 많습니다.

구별·업종 조합에서는 강남구 한식 3,917개, 마포구 한식 2,835개, 송파구 한식 2,626개, 강남구 커피점/카페 2,253개가 상위에 있습니다. 전체 점포수와 업종별 밀집도를 함께 보아야 한다는 뜻입니다.

따라서 상권 조사 담당자는 강남구, 마포구, 송파구를 먼저 비교하고, 각 구에서 한식과 커피점/카페의 경쟁 강도를 확인한 뒤 후보지를 좁히는 방식이 적절합니다. 보고서에는 구별·업종 조합 막대그래프 1개와 상위 10개 표 1개만 남기고 원자료 전체 표는 부록으로 분리하겠습니다.
```

## 평가 기준 해설

상:

```text
발견, 시사점, 액션이 분리되어 있고, 발견에 실제 숫자가 2개 이상 들어 있다. 차트 선택 이유가 결론과 연결되어 있으며, 동료 피드백을 받아 제목이나 액션을 구체적으로 고쳤다.
```

중:

```text
보고서 형식은 갖추었고 숫자도 일부 들어 있지만, 시사점이나 액션이 다소 추상적이다. 예를 들어 "추가 분석 필요"에서 멈추면 중 수준이다.
```

하:

```text
차트나 표는 있으나 결론이 없고, 실제 숫자가 부족하거나 S19 결과물과 연결되지 않는다. 동료 피드백 반영 흔적도 없다.
```


핵심 코드:

```python
from pathlib import Path

import pandas as pd


ROOT = Path(__file__).resolve().parents[1]
CAFE_RESTAURANT_PATH = (
    ROOT / "02_datasets/real_korean/cafe_restaurant/seoul_commercial_food_stores_202109_github.csv"
)

cafe = pd.read_csv(CAFE_RESTAURANT_PATH)

top_gu = (
    cafe.groupby("시군구명")
    .size()
    .sort_values(ascending=False)
    .head(3)
    .reset_index(name="점포수")
)

top_gu_category = (
    cafe.groupby(["시군구명", "상권업종중분류명"])
    .size()
    .sort_values(ascending=False)
    .head(4)
    .reset_index(name="점포수")
)

print(len(cafe))
print(top_gu)
print(top_gu_category)
```
