# S08. 모범답안

## 미션 해설

미션은 같은 설문 데이터 또는 같은 표본 18행을 두고 프롬프트 A/B/C를 비교하는 것이다. 정답 문장은 하나로 고정되지 않지만, 좋은 답안은 다음 조건을 충족해야 한다.

1. 데이터 맥락을 설명한다.
2. AI가 해야 할 질문을 좁힌다.
3. 결과 표의 컬럼을 지정한다.
4. 원문 근거와 응답자 ID를 요구한다.
5. 검토 필요 행을 찾게 한다.
6. 표본 분석과 전체 분석을 구분한다.

## 예시 답안: 프롬프트 A

```text
이 설문 데이터를 분석해줘.
```

평가:

프롬프트 A는 의도적으로 부족한 프롬프트다. 맥락, 질문, 출력형식이 모두 빠져 있다. AI가 요약문을 만들 수는 있지만, 어떤 기준으로 주제를 나눴는지 확인하기 어렵고 결과를 엑셀에 붙여 넣어 다시 집계하기도 어렵다.

## 예시 답안: 프롬프트 B

```text
너는 고객경험 분석가다.
데이터는 온라인 쇼핑 서비스의 고객 설문이다. 한 행은 응답 1건이다.
주요 컬럼은 respondent_id, age_group, channel, nps, satisfaction, free_text다.

목표는 free_text를 읽고 고객 의견을 운영 개선에 쓸 수 있게 정리하는 것이다.
각 응답을 주제와 감성으로 분류해줘.

주제는 6-8개로 만들되 너무 세분화하지 마라.
감성은 긍정, 부정, 혼합, 중립 중 하나로 정하라.
NPS와 free_text의 분위기가 어긋나면 혼합 또는 검토 필요로 표시하라.

출력은 표로 작성하라.
표 컬럼은 respondent_id, topic, sentiment, evidence, confidence, needs_review로 해라.
evidence에는 판단 근거가 된 원문 일부를 넣어라.
confidence는 높음, 중간, 낮음 중 하나로 표시하라.
```

평가:

프롬프트 B는 수업 기준으로 통과 가능한 프롬프트다. 맥락, 질문, 출력형식이 들어 있고 원문 근거도 요구한다. 다만 코드북을 먼저 확인하지 않고 바로 분류하므로 주제 기준이 마음에 들지 않을 때 전체 결과를 다시 받아야 한다.

## 예시 답안: 프롬프트 C

```text
아래 설문 데이터를 분석하되, 한 번에 결론을 내지 말고 3단계로 진행해줘.

데이터 맥락:
- 온라인 쇼핑 서비스의 고객 설문이다.
- 한 행은 응답 1건이다.
- 주요 컬럼은 respondent_id, age_group, channel, nps, satisfaction, free_text다.
- 목표는 주관식 의견을 운영 개선에 쓸 수 있게 분류하는 것이다.

1단계:
free_text를 분류할 주제 코드북을 먼저 제안해줘.
주제는 6-8개로 제한하고, 각 주제의 정의와 포함 예시를 적어줘.
서로 겹치기 쉬운 주제는 구분 기준도 적어줘.
이 단계에서는 행별 분류를 하지 마라.

2단계:
내가 코드북을 확인한 뒤, 그 코드북으로 각 행을 분류해줘.
출력 표 컬럼은 respondent_id, topic, sentiment, evidence, confidence, needs_review로 해라.
sentiment는 긍정, 부정, 혼합, 중립 중 하나로 정하라.
NPS와 free_text가 어긋나면 혼합 또는 needs_review=예로 표시하라.
원문에 없는 원인이나 고객 의도는 추측하지 마라.

3단계:
분류 결과를 주제별 응답수, 비중, 평균 NPS, 평균 satisfaction, 대표 근거문구로 요약해줘.
마지막에는 사람이 다시 확인해야 할 행 5개를 respondent_id와 이유로 정리해줘.

파일이 아니라 표본 데이터만 사용한 경우에는 모든 결론 앞에 "표본 기준"이라고 표시해줘.
```

평가:

프롬프트 C가 가장 안정적이다. 코드북을 먼저 만들게 하므로 분류 기준을 검토할 수 있고, 이후 같은 기준으로 행별 분류와 요약을 진행할 수 있다. 또한 `respondent_id`, `evidence`, `needs_review`가 있어 원본 대조와 사람 검토가 가능하다.

## A/B/C 비교표 예시

| 항목 | 프롬프트 A | 프롬프트 B | 프롬프트 C |
|---|---|---|---|
| 맥락이 충분한가 | 부족 | 있음 | 충분 |
| 질문이 구체적인가 | 부족 | 구체적 | 단계별로 구체적 |
| 출력형식이 표로 고정되는가 | 불안정 | 고정 | 고정 |
| 원문 근거가 포함되는가 | 보장 안 됨 | 포함 요청 | 포함 요청 |
| 검토 필요 행을 찾는가 | 보장 안 됨 | 일부 가능 | 명시적으로 요청 |
| 업무에 바로 쓰기 좋은가 | 낮음 | 보통 이상 | 가장 좋음 |

정리 문장 예시:

`내가 선택한 최종 프롬프트는 C이며, 이유는 코드북을 먼저 확인한 뒤 같은 기준으로 행별 분류와 요약을 진행할 수 있기 때문이다. 특히 respondent_id와 evidence 열이 있어 원본과 대조할 수 있다.`

## 실습 코드

아래와 같은 코드를 `/mnt/d/ai-data-analysis-course/06_answers/S08_code.py`에 저장했다.

```python
from pathlib import Path

import pandas as pd


ROOT = Path("/mnt/d/ai-data-analysis-course")
SURVEY_CLEAN = ROOT / "02_datasets/survey_responses/survey_responses_clean.csv"
SURVEY_DIRTY = ROOT / "02_datasets/survey_responses/survey_responses_dirty.csv"
MOVIES = ROOT / "02_datasets/real_korean/movie_box/dacon_movies_train.csv"

EXPECTED_SURVEY_COLUMNS = [
    "respondent_id",
    "age_group",
    "gender",
    "nps",
    "satisfaction",
    "channel",
    "free_text",
    "submitted_at",
]


def nps_group(score: int) -> str:
    if score <= 6:
        return "비추천(0-6)"
    if score <= 8:
        return "중립(7-8)"
    return "추천(9-10)"


def classify_topic(text: str) -> str:
    rules = [
        ("배송/픽업", ["배송", "포장", "픽업", "기사님"]),
        (
            "고객지원/응대",
            ["고객센터", "상담", "문의", "답변", "처리", "교환", "환불", "반품", "응대", "문제 발생"],
        ),
        ("결제/혜택", ["결제", "쿠폰", "할인", "포인트", "혜택", "회원", "가격"]),
        ("상품정보/품질", ["상품 설명", "상품 비교", "품질", "실제 구성", "상세 페이지", "사진"]),
        ("매장/오프라인", ["매장", "오프라인", "재고", "헛걸음"]),
        ("알림/마케팅", ["알림", "푸시", "이벤트", "프로모션", "추천 상품"]),
        (
            "앱/사용성",
            ["모바일", "앱", "화면", "검색", "필터", "로딩", "메뉴", "디자인", "주소", "자동완성", "가입", "이전 구매", "예약", "정보", "글자"],
        ),
    ]
    for topic, keywords in rules:
        if any(keyword in text for keyword in keywords):
            return topic
    return "기타"


def classify_text_sentiment(text: str) -> str:
    negative_or_improve = [
        "불편",
        "헷갈",
        "오류",
        "늦",
        "오래",
        "길",
        "복잡",
        "부담",
        "개선",
        "재시도",
        "어렵",
        "달라",
        "많아서",
        "더 좋",
        "있으면",
        "아쉬",
        "표시되면",
        "바꿨습니다",
    ]
    positive = [
        "좋",
        "편리",
        "정확",
        "만족",
        "쉬",
        "친절",
        "명확",
        "빨",
        "안심",
        "신뢰",
        "깔끔",
        "직관",
        "잘",
        "도움",
        "부담이 없",
        "해결",
        "일치",
    ]
    has_negative = any(word in text for word in negative_or_improve)
    has_positive = any(word in text for word in positive)
    if has_negative and has_positive:
        return "혼합/개선"
    if has_negative:
        return "부정/개선"
    if has_positive:
        return "긍정"
    return "중립"


def markdown_table(df: pd.DataFrame) -> str:
    cols = list(df.columns)
    lines = [
        "| " + " | ".join(cols) + " |",
        "| " + " | ".join(["---"] * len(cols)) + " |",
    ]
    for _, row in df.iterrows():
        lines.append("| " + " | ".join(str(row[col]).replace("|", "/") for col in cols) + " |")
    return "\n".join(lines)


def main() -> None:
    survey = pd.read_csv(SURVEY_CLEAN)
    dirty = pd.read_csv(SURVEY_DIRTY)
    movies = pd.read_csv(MOVIES)

    assert list(survey.columns) == EXPECTED_SURVEY_COLUMNS
    assert survey.shape == (1500, 8)
    assert dirty.shape == (1510, 8)
    assert movies.shape == (600, 12)
    assert survey["free_text"].isna().sum() == 0

    survey = survey.copy()
    survey["nps_group"] = survey["nps"].apply(nps_group)
    survey["topic_rule"] = survey["free_text"].apply(classify_topic)
    survey["text_sentiment_rule"] = survey["free_text"].apply(classify_text_sentiment)

    sample_parts = []
    for group in ["비추천(0-6)", "중립(7-8)", "추천(9-10)"]:
        part = survey.loc[survey["nps_group"] == group].sample(n=6, random_state=8)
        sample_parts.append(part)
    prompt_sample = pd.concat(sample_parts).sort_values(["nps", "respondent_id"])

    topic_summary = (
        survey.groupby("topic_rule")
        .agg(
            응답수=("respondent_id", "count"),
            평균NPS=("nps", "mean"),
            평균만족도=("satisfaction", "mean"),
        )
        .round({"평균NPS": 2, "평균만족도": 2})
        .sort_values(["응답수", "평균NPS"], ascending=[False, True])
        .reset_index()
        .rename(columns={"topic_rule": "주제"})
    )

    sentiment_summary = (
        survey["text_sentiment_rule"]
        .value_counts()
        .rename_axis("주관식_감성_휴리스틱")
        .reset_index(name="응답수")
    )

    dirty_issues = {
        "dirty_rows": len(dirty),
        "duplicated_respondent_id": int(dirty.duplicated("respondent_id").sum()),
        "missing_nps": int(dirty["nps"].isna().sum()),
        "missing_free_text": int(dirty["free_text"].isna().sum()),
        "non_numeric_satisfaction": int(pd.to_numeric(dirty["satisfaction"], errors="coerce").isna().sum()),
        "age_group_variants_for_20s": int(dirty["age_group"].isin(["20s", "이십대"]).sum()),
    }

    genre_summary = (
        movies.groupby("genre")
        .agg(영화수=("title", "count"), 평균관객수=("box_off_num", "mean"))
        .round({"평균관객수": 0})
        .astype({"평균관객수": "int64"})
        .sort_values("영화수", ascending=False)
        .head(5)
        .reset_index()
    )

    print("pandas version:", pd.__version__)
    print("survey clean shape:", survey.shape[0], "rows x", 8, "columns")
    print("survey dirty shape:", dirty.shape[0], "rows x", dirty.shape[1], "columns")
    print("dacon movies train shape:", movies.shape[0], "rows x", movies.shape[1], "columns")

    print("\n[dirty issue counts]")
    for key, value in dirty_issues.items():
        print(f"{key}: {value}")

    print("\n[NPS group counts]")
    print(survey["nps_group"].value_counts().reindex(["비추천(0-6)", "중립(7-8)", "추천(9-10)"]).to_string())

    print("\n[topic summary by simple rules]")
    print(topic_summary.to_string(index=False))

    print("\n[text sentiment summary by simple rules]")
    print(sentiment_summary.to_string(index=False))

    print("\n[prompt sample 18 rows]")
    sample_cols = ["respondent_id", "age_group", "channel", "nps", "satisfaction", "free_text"]
    print(markdown_table(prompt_sample[sample_cols]))

    print("\n[optional movie genre top 5]")
    print(genre_summary.to_string(index=False))


if __name__ == "__main__":
    main()
```
