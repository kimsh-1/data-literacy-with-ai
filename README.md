# 데이터 리터러시 with AI

엑셀은 다뤄봤지만 데이터 분석과 코딩, 통계는 처음인 사람을 위한 데이터 분석 입문 과정입니다. 코드를 직접 짜지 못해도 AI를 지렛대 삼아 엑셀에서 파이썬, BI 대시보드까지 한 바퀴를 스스로 돌리는 것을 목표로 합니다.

## 이 과정이 향하는 곳

핵심은 도구를 바꿔 가며 같은 질문에 답하는 흐름입니다.

```
엑셀 → AI → 파이썬 → BI 대시보드
```

같은 데이터를 네 도구로 번갈아 다루기 때문에, 배우는 사람은 도구가 바뀔 때 무엇이 쉬워지고 무엇이 달라지는지에만 집중하면 됩니다. AI가 분석을 대신 해 주더라도, 무엇을 물어보고 그 답이 맞는지 검증하는 눈은 사람이 가진다는 원칙을 처음부터 끝까지 강조합니다.

## 대상

- 엑셀 기본 조작(셀 입력, 간단한 함수)은 해 본 완전 입문자
- 비개발자, 실무에서 데이터를 다루기 시작한 사람

## 구성 — 6개 파트, 20차시

| 파트 | 주제 | 차시 |
|---|---|---|
| 0 | 오리엔테이션과 데이터 마인드셋 | S01 |
| 1 | 엑셀에서 데이터 감각 잡기 | S02~S04 |
| 2 | 숫자에 속지 않는 통계 직관 | S05~S06 |
| 3 | AI로 분석하기 (노코드) | S07~S09 |
| 4 | 파이썬과 Pandas 제대로 | S10~S14 |
| 5 | BI 대시보드 (Power BI / Looker) | S15~S18 |
| 6 | 종합 캡스톤 | S19~S20 |

## 폴더 구조

- `00_PLAN/` — 커리큘럼 설계, 차시 스펙, 데이터 카탈로그
- `01_curriculum/` — 차시별 커리큘럼 문서
- `03_handouts/` — 수강생 핸드아웃 (따라하기와 미션)
- `04_scripts/` — 강사 대본
- `06_answers/` — 모범답안과 실습 코드
- `02_datasets/` — 예시 데이터와 생성 스크립트
- `site/` — 수업 소개 웹사이트 (정적, 배포용)
- `assets/` — 이미지와 소개 영상

## 데이터

두 종류를 씁니다.

- **합성 데이터**(이 저장소에 포함): 카페 매출, 쇼핑몰 주문(3개 테이블), 고객 설문, 인사 데이터. 난이도를 통제하려고 직접 만들었고, 생성 스크립트도 `02_datasets/_generators/`에 함께 둡니다.
- **실제 한국 공개 데이터**(저장소에는 미포함): 따릉이, 지하철 승하차, 아파트 실거래, 상가업소, 기상 관측 등 52종. 정부·공개 데이터의 재배포 조건을 존중해 원본은 담지 않고, 출처와 내려받는 방법을 `00_PLAN/REAL_DATA_CATALOG.md`에 정리했습니다.

## 소개 웹사이트

`site/index.html`은 커리큘럼과 실습 데이터를 한 화면에서 훑어보는 소개 사이트입니다. 별도 빌드 없이 정적 호스팅에 바로 올릴 수 있습니다.

## 라이선스

수업 자료(문서·코드·합성 데이터)는 자유롭게 활용하되, 실제 공개 데이터는 각 출처의 이용 약관을 따릅니다.
