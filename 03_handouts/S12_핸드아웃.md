## 따라하기 2: 문자열, 타입, 결측값 정리

이제 계산 가능한 표로 만들 차례다. 원본 `df`는 그대로 두고 `cleaned`라는 복사본을 만든다.

```python
cleaned = df.copy()

cleaned["store"] = cleaned["store"].astype("string").str.strip().str.replace(" ", "", regex=False)
cleaned["category"] = cleaned["category"].astype("string").str.strip()
cleaned["item"] = (
    cleaned["item"]
    .astype("string")
    .str.strip()
    .replace({
        "아메리카": "아메리카노",
        "Americano": "아메리카노",
        "아메 리카노": "아메리카노",
    })
)

cleaned["payment"] = cleaned["payment"].astype("string").str.strip().fillna("미확인")
cleaned["member"] = cleaned["member"].astype("boolean").fillna(False)
```

해석:

- `store`, `category`, `item`은 그룹명이 흔들리지 않게 정리한다.
- `payment` 빈칸은 주문을 버리지 않고 `미확인`으로 채운다.
- `member` 빈칸은 이번 실습 기준으로 비회원 `False`로 채운다.

## 따라하기 4: 중복, 결측, 이상치 제거

```python
before = len(cleaned)
cleaned = cleaned.drop_duplicates(subset=["order_id"], keep="first").copy()
print("중복 제거 행수:", before - len(cleaned))

before = len(cleaned)
cleaned = cleaned.dropna(
    subset=["date", "store", "category", "item", "unit_price", "qty", "amount"]
).copy()
print("필수값 결측으로 제거한 행수:", before - len(cleaned))
```

기대 결과:

- 중복 `order_id` 제거: 15행
- 필수값 결측으로 제거: 0행

이번 데이터에서는 필수값 결측으로 삭제되는 행이 없다. 그래도 `dropna(subset=...)` 패턴은 중요하다. 실제 업무에서는 날짜나 금액이 비어 있는 행이 자주 나온다.

### IQR로 수량 이상치 찾기

```python
q1 = cleaned["qty"].quantile(0.25)
q3 = cleaned["qty"].quantile(0.75)
iqr = q3 - q1
lower = q1 - 1.5 * iqr
upper = q3 + 1.5 * iqr

print(q1, q3, iqr, lower, upper)

qty_outlier = ~cleaned["qty"].between(lower, upper)
cleaned.loc[qty_outlier, ["order_id", "date", "store", "item", "qty", "amount"]]
```

기대 결과:

- Q1: 1.0
- Q3: 2.0
- IQR: 1.0
- 정상범위: -0.5 ~ 3.5
- `qty=999`인 2행이 이상치로 잡힌다.

이상치와 음수 금액을 제거한다.

```python
cleaned = cleaned.loc[~qty_outlier].copy()
cleaned = cleaned.loc[cleaned["amount"] > 0].copy()

for col in ["order_id", "hour", "unit_price", "qty", "amount"]:
    cleaned[col] = cleaned[col].astype("int64")

cleaned["expected_amount"] = cleaned["unit_price"] * cleaned["qty"]
cleaned["date_month"] = cleaned["date"].dt.to_period("M").astype("string")
```
