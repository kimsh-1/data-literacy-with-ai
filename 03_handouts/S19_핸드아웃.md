## 따라하기 2. AI 요약 만들기

AI 요약에는 원자료 CSV 전체를 올리지 않는다. 파이썬이 만든 집계 텍스트만 사용한다.

먼저 입력 텍스트를 확인한다.

```bash
sed -n '1,160p' 06_answers/S19_outputs/S19_ai_summary_input.txt
```

AI 도구에 아래 프롬프트를 그대로 붙여넣고, 그 아래에 `S19_ai_summary_input.txt` 내용을 붙인다.

```text
너는 데이터분석 입문 수업의 조교다.
아래 집계 결과만 근거로 서울 음식 업종 상권 요약을 작성하라.

조건:
- 5문장 이내로 작성한다.
- 가장 점포 수가 많은 구와 상위 업종을 포함한다.
- "매출이 높다", "창업하기 좋다"처럼 데이터에 없는 결론은 쓰지 않는다.
- 마지막 문장은 이 데이터로 판단할 수 없는 한계를 적는다.

[집계 결과]
```

유료 파일 업로드 기능이 없어도 괜찮다. 이 단계는 파일을 올릴 필요 없이 텍스트를 복사해 붙이는 것으로 충분하다.

요약 예시는 다음과 같은 구조면 된다.

```text
서울 음식 업종 점포는 강남구에 가장 많이 분포하며, 다음으로 마포구와 송파구가 많다. 업종 중분류 기준으로는 한식이 가장 많고, 커피점/카페가 그 다음으로 많다. 점포 수 상위 구에서도 한식 비중이 대체로 크며, 카페 비율은 구마다 차이가 있다. 이 결과는 점포 수 기준의 분포를 보여주지만 매출, 수익성, 유동인구는 포함하지 않는다.
```

## 따라하기 3. 파이썬 그래프 확인

다음 파일을 연다.

```text
06_answers/S19_outputs/S19_python_top10_gu_chart.png
```

그래프를 보고 아래 질문에 답한다.

1. 점포 수가 가장 많은 구는 어디인가?
2. 1위 구와 2위 구의 차이가 큰가?
3. 그래프 옆에 표시된 카페 비율은 무엇을 보조로 설명하는가?
4. 이 그래프만 보고 말하면 안 되는 결론은 무엇인가?

정답 방향은 다음과 같다.

- 1위는 강남구다.
- 강남구 12,095개와 마포구 7,396개의 차이가 크다.
- 카페 비율은 해당 구의 음식 업종 중 커피점/카페가 차지하는 비중이다.
- 매출, 임대료, 수익성, 창업 성공 가능성은 이 그래프만으로 말할 수 없다.

## 따라하기 4. Looker Studio로 BI 1페이지 만들기

Mac 수강생은 이 경로를 기본으로 사용한다. Windows 수강생도 웹에서 진행하려면 같은 방법을 사용한다.

1. Looker Studio에 접속한다.
2. 빈 보고서를 만든다.
3. 데이터 소스로 `파일 업로드` 또는 Google Sheets 업로드를 선택한다.
4. 아래 CSV를 연결한다.

```text
Mac/WSL: /mnt/d/ai-data-analysis-course/06_answers/S19_outputs/S19_bi_input_gu_category.csv
Windows: D:\ai-data-analysis-course\06_answers\S19_outputs\S19_bi_input_gu_category.csv
```

5. 필드 형식을 확인한다.

| 필드 | 권장 형식 |
|---|---|
| `시군구명` | 텍스트 |
| `상권업종중분류명` | 텍스트 |
| `점포수` | 숫자 |
| `구내비율` | 퍼센트 |
| `전체대비비율` | 퍼센트 |

6. 보고서 제목을 추가한다.

```text
서울 음식 업종 점포 분포 대시보드
```

7. 스코어카드 2개를 만든다.

| 스코어카드 | 측정항목 |
|---|---|
| 총 점포 수 | `점포수`의 합계 |
| 시군구 수 | `시군구명`의 고유 개수 |

8. 구별 점포 수 막대그래프를 만든다.

| 설정 | 값 |
|---|---|
| 차트 종류 | 막대그래프 |
| 측정기준 | `시군구명` |
| 측정항목 | `점포수` 합계 |
| 정렬 | `점포수` 내림차순 |

9. 구별 업종 구성 그래프를 만든다.

| 설정 | 값 |
|---|---|
| 차트 종류 | 누적 막대그래프 또는 표 |
| 측정기준 | `시군구명` |
| 세부 측정기준 | `상권업종중분류명` |
| 측정항목 | `점포수` 합계 |

10. 필터 컨트롤을 추가한다.

| 설정 | 값 |
|---|---|
| 필터 기준 | `상권업종중분류명` |
| 목적 | 한식, 커피점/카페, 분식 등을 선택해 구별 분포 비교 |

11. 해석 메모를 텍스트 상자로 추가한다.

```text
점포 수 기준으로는 강남구가 가장 많다. 이 대시보드는 점포 분포를 보여주며 매출이나 수익성을 직접 설명하지 않는다.
```

## Power BI Desktop 대안

Power BI Desktop은 Windows 전용이다. Windows 수강생은 아래 순서로 진행할 수 있다.

1. Power BI Desktop을 연다.
2. `데이터 가져오기`에서 `텍스트/CSV`를 선택한다.
3. `06_answers/S19_outputs/S19_bi_input_gu_category.csv`를 연다.
4. `점포수`는 정수, `구내비율`과 `전체대비비율`은 소수 또는 백분율로 둔다.
5. 카드, 막대그래프, 누적 막대그래프, 슬라이서를 1페이지에 배치한다.
6. 슬라이서 필드는 `상권업종중분류명`으로 둔다.
