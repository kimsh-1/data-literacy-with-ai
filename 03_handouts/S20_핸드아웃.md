## 따라하기: 1페이지 보고서 만들기

목표는 차트를 새로 잔뜩 만드는 것이 아니라, 이미 만든 분석 결과를 보고서 한 장으로 옮기는 것이다.

### 1단계. 결론 제목 쓰기

보고서 제목은 주제가 아니라 결론이어야 한다.

나쁜 제목:

```text
서울 상권 분석
```

좋은 제목:

```text
서울 음식점·카페 상권은 강남구에 가장 집중되어 있으며, 구별 우선순위는 한식과 커피점/카페 조합부터 확인해야 한다.
```

문장이 길어도 상관없다. 제목만 읽고도 결론이 어느 쪽인지 짐작할 수 있어야 한다.

### 2단계. 발견 2-3개 쓰기

S19 결과물이나 `S20_code.py` 실행 결과에서 숫자를 가져온다.

강사 예시:

```text
분석 대상은 서울 음식점·카페 상권 데이터 119,158행이다.
구별 점포수는 강남구 12,095개, 마포구 7,396개, 송파구 7,106개 순으로 많다.
구별·업종 조합은 강남구 한식 3,917개, 마포구 한식 2,835개, 송파구 한식 2,626개, 강남구 커피점/카페 2,253개가 상위에 온다.
```

"오를 것이다", "무조건 좋다", "확실하다" 같은 표현은 발견에 넣지 않도록 조심한다. 발견에는 실제로 확인한 숫자만 담는다.

### 3단계. 시각화 1개 고르기

상권 케이스에서는 아래 중 하나를 고른다.

1. S19에서 만든 구별 점포수 상위 10개 막대그래프
2. BI에서 만든 구별·업종별 점포수 비교 화면
3. `06_answers/S20_outputs/s20_cafe_top_gu_category.csv`를 엑셀로 열어 만든 구별·업종 조합 상위 10개 막대그래프

추천은 구별·업종 조합 상위 10개 막대그래프다. 어느 구와 업종 조합을 먼저 봐야 할지 가장 직접적으로 보여주기 때문이다.

차트 제목 예시:

```text
강남구 한식과 강남구 커피점/카페가 상권 조사 우선 후보에 들어간다
```

차트 아래 해석 문장 예시:

```text
강남구 한식은 3,917개, 강남구 커피점/카페는 2,253개로 상위권이다. 전체 점포수가 많은 구라도 업종별 밀집도가 다르므로 구와 업종을 함께 비교해야 한다.
```

### 4단계. 시사점 쓰기

발견을 업무 의미로 바꾼다.

```text
상권 후보를 고를 때는 전체 점포수가 많은 구를 먼저 보되, 실제 실행 우선순위는 업종별 밀집도까지 함께 보아야 한다. 강남구는 전체 점포수와 한식·커피점/카페 점포수가 모두 높으므로 1차 조사 후보로 적합하다.
```

### 5단계. 액션 2-3개 쓰기

보고서를 읽은 사람이 다음에 할 행동을 쓴다.

```text
1. 상권 조사 담당자는 강남구, 마포구, 송파구를 1차 비교 대상으로 잡는다.
2. 각 구는 한식과 커피점/카페 점포수를 먼저 비교하고, 경쟁 강도가 높은 조합을 표시한다.
3. 의사결정 회의에는 구별 점포수 막대그래프 1개와 구별·업종 상위 10개 표 1개만 넣고, 원자료 전체 표는 부록으로 분리한다.
```

### 6단계. 한계와 주의점 쓰기

좋은 보고서는 한계를 숨기지 않는다.

```text
이 분석은 2021년 서울 음식점·카페 상권 데이터의 점포수만 사용했다. 실제 출점 판단에는 임대료, 유동인구, 매출, 배달 수요, 폐업률, 접근성 같은 추가 정보가 필요하다.
```

### 7단계. 3분 발표 원고 만들기

아래 구조로 말한다.

```text
1분: 결론
서울 음식점·카페 상권은 강남구에 가장 집중되어 있으며, 구별 우선순위는 한식과 커피점/카페 조합부터 확인해야 합니다.

1분: 근거
119,158행을 집계한 결과, 구별 점포수는 강남구 12,095개, 마포구 7,396개, 송파구 7,106개 순입니다. 구별·업종 조합에서는 강남구 한식 3,917개와 강남구 커피점/카페 2,253개가 상위권에 들어갑니다.

1분: 액션
따라서 상권 조사 담당자는 강남구, 마포구, 송파구를 먼저 비교하고, 각 구에서 한식과 커피점/카페의 경쟁 강도를 확인한 뒤 후보지를 좁히는 방식이 적절합니다.
```

## 직접 해보기: 동료 피드백 반영

본인의 S19 결과물을 1페이지 보고서로 바꾼다. 기본 실습은 상권의 구별·업종별 집계 결과를 사용한다. 원자료 119,158행을 그대로 차트에 넣지 말고, 구별 또는 업종별로 집계한 상위 N개만 사용한다. 아파트 결과물을 대안 예시로 쓴다면 산점도와 추세선의 방향, 상관계수, 같은 면적대의 가격 범위까지만 다룬다.

### 제출물

1. 1페이지 보고서
2. 3분 발표 원고
3. 동료 피드백 2개와 반영 결과

### 보고서 템플릿

```text
제목:

핵심 발견:
1.
2.
3.

핵심 시각화:
- 차트 이름:
- 왜 이 차트를 골랐는가:

시사점:

권장 액션:
1.
2.
3.

주의점:

동료 피드백과 반영:
1. 받은 피드백:
   반영 내용:
2. 받은 피드백:
   반영 내용:
```

### 동료 피드백 기준

짝은 아래 질문에 답한다.

```text
1. 제목만 읽어도 결론을 알 수 있는가?
2. 발견 문장에 실제 숫자가 들어 있는가?
3. 시사점이 발견과 연결되는가?
4. 액션이 구체적인 행동으로 쓰였는가?
5. 차트가 결론을 직접 보여주는가?
```

## 제출 기준

- 발견, 시사점, 액션이 구분되어 있다.
- 발견에는 실제 숫자가 2개 이상 들어 있다.
- 차트는 결론을 설명하는 1개를 골랐다.
- 3분 발표 원고가 결론, 근거, 액션 순서로 되어 있다.
- 동료 피드백 2개 이상을 받고 최소 1개 이상 반영했다.
