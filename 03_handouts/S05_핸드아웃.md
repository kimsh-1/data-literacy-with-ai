## 오늘의 목표

이번 실습에서는 직원 연봉 데이터로 부서별 분포를 비교한다. 평균만 구하고 끝내지 않고 중앙값, 최빈값, 범위, 표준편차까지 함께 구한 뒤 박스플롯으로 분포를 확인한다. 마지막에는 카페 주문 데이터에서 평균이 보통 주문금액을 과장해 보이게 만드는 사례를 찾는다.

## 따라하기 1. 부서별 연봉 요약표 만들기

### 1단계. 파일 열고 컬럼 확인하기

1. `hr_employees_clean.xlsx`를 연다.
2. `employees` 시트를 확인한다.
3. 다음 컬럼을 찾는다.

| 컬럼 | 의미 |
|---|---|
| `dept` | 부서 |
| `salary` | 연봉 |

이 파일은 1행이 제목이고, 실제 데이터는 2행부터 1,201행까지 있다. `dept`는 B열, `salary`는 G열이다.

### 2단계. 요약 시트 만들기

1. 새 시트를 만든다.
2. 시트 이름을 `dept_salary_summary`로 바꾼다.
3. A1:O1에 아래 제목을 입력한다.

| 셀 | 제목 |
|---|---|
| A1 | `dept` |
| B1 | `count` |
| C1 | `mean` |
| D1 | `median` |
| E1 | `mode` |
| F1 | `min` |
| G1 | `max` |
| H1 | `range` |
| I1 | `std_dev` |
| J1 | `q1` |
| K1 | `q3` |
| L1 | `iqr` |
| M1 | `lower_fence` |
| N1 | `upper_fence` |
| O1 | `outlier_count` |

4. A2:A6에 부서명을 입력한다.

```text
개발본부
데이터전략팀
영업본부
인사총무팀
고객운영팀
```

### 3단계. 직원 수와 중심값 계산하기

B2 셀에 입력한다.

```excel
=COUNTIF(employees!$B$2:$B$1201,A2)
```

C2 셀에 입력한다.

```excel
=AVERAGEIF(employees!$B$2:$B$1201,A2,employees!$G$2:$G$1201)
```

D2 셀에 입력한다.

```excel
=MEDIAN(FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A2))
```

E2 셀에 입력한다.

```excel
=MODE.SNGL(FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A2))
```

B2:E2를 6행까지 채운다.

뜻은 다음과 같다.

- `count`: 부서별 직원 수
- `mean`: 부서별 평균 연봉
- `median`: 부서별 중앙 연봉
- `mode`: 부서별 가장 자주 나오는 연봉

### 4단계. 범위와 표준편차 계산하기

F2 셀에 입력한다.

```excel
=MINIFS(employees!$G$2:$G$1201,employees!$B$2:$B$1201,A2)
```

G2 셀에 입력한다.

```excel
=MAXIFS(employees!$G$2:$G$1201,employees!$B$2:$B$1201,A2)
```

H2 셀에 입력한다.

```excel
=G2-F2
```

I2 셀에 입력한다.

```excel
=STDEV.S(FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A2))
```

F2:I2를 6행까지 채운다.

뜻은 다음과 같다.

- `min`: 최저 연봉
- `max`: 최고 연봉
- `range`: 최고 연봉과 최저 연봉의 차이
- `std_dev`: 평균 주변에서 값이 흩어진 정도

### 5단계. 사분위수와 이상치 기준 계산하기

J2 셀에 입력한다.

```excel
=QUARTILE.INC(FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A2),1)
```

K2 셀에 입력한다.

```excel
=QUARTILE.INC(FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A2),3)
```

L2 셀에 입력한다.

```excel
=K2-J2
```

M2 셀에 입력한다.

```excel
=J2-1.5*L2
```

N2 셀에 입력한다.

```excel
=K2+1.5*L2
```

O2 셀에 입력한다.

```excel
=COUNTIFS(employees!$B$2:$B$1201,A2,employees!$G$2:$G$1201,"<"&M2)+COUNTIFS(employees!$B$2:$B$1201,A2,employees!$G$2:$G$1201,">"&N2)
```

J2:O2를 6행까지 채운다.

`FILTER` 함수가 없는 엑셀을 사용하는 경우에는 중앙값, 최빈값, 표준편차, 사분위수 계산을 다음 방식으로 대체한다.

1. `employees` 시트의 `dept`, `salary` 열을 빈 보조 영역에 복사한다.
2. 보조 영역을 `dept` 기준으로 정렬해 같은 부서의 `salary`가 연속되게 만든다. 또는 `dept` 필터로 한 부서만 보이게 한 뒤 보이는 `salary` 값을 보조열에 복사한다.
3. 각 부서의 `salary` 범위를 직접 참조해 `=MEDIAN(범위)`, `=MODE.SNGL(범위)`, `=STDEV.S(범위)`, `=QUARTILE.INC(범위,1)`, `=QUARTILE.INC(범위,3)`을 입력한다.
4. 예를 들어 개발본부 `salary`가 보조열 Q2:Q371에 있으면 D2에는 `=MEDIAN($Q$2:$Q$371)`, E2에는 `=MODE.SNGL($Q$2:$Q$371)`, I2에는 `=STDEV.S($Q$2:$Q$371)`, J2에는 `=QUARTILE.INC($Q$2:$Q$371,1)`, K2에는 `=QUARTILE.INC($Q$2:$Q$371,3)`을 입력한다.

이상치 기준은 입문 단계에서 다음처럼 이해하면 된다.

```text
IQR = Q3 - Q1
아래쪽 이상치 후보 = Q1 - 1.5 × IQR 보다 작은 값
위쪽 이상치 후보 = Q3 + 1.5 × IQR 보다 큰 값
```

## 따라하기 2. 부서별 연봉 박스플롯 만들기

### 1단계. 박스플롯용 시트 만들기

1. 새 시트를 만든다.
2. 시트 이름을 `dept_boxplot`으로 바꾼다.
3. A1:E1에 부서명을 입력한다.

| 셀 | 제목 |
|---|---|
| A1 | `개발본부` |
| B1 | `데이터전략팀` |
| C1 | `영업본부` |
| D1 | `인사총무팀` |
| E1 | `고객운영팀` |

### 2단계. 부서별 연봉을 열로 펼치기

A2 셀에 입력한다.

```excel
=FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A$1)
```

A2 셀의 수식을 E2까지 복사한다.

각 부서의 연봉이 아래쪽으로 자동으로 펼쳐진다. 부서마다 직원 수가 다르므로 열마다 채워지는 길이가 다를 수 있다.

`FILTER` 함수가 없는 엑셀을 사용하는 경우에는 다음 방식으로 대체한다.

1. `employees` 시트에서 `dept` 열에 필터를 건다.
2. `개발본부`만 선택한다.
3. 보이는 `salary` 값을 복사해 `dept_boxplot` 시트 A열에 붙여넣는다.
4. 나머지 부서도 B열부터 E열까지 같은 방식으로 붙여넣는다.

### 3단계. 박스플롯 삽입하기

1. `dept_boxplot` 시트에서 A1:E371 범위를 선택한다.
2. 상단 메뉴에서 `삽입`을 클릭한다.
3. `통계 차트 삽입`을 클릭한다.
4. `박스플롯`을 선택한다.
5. 차트 제목을 `부서별 연봉 분포`로 바꾼다.

차트에서 따로 찍힌 점은 이상치 후보로 본다. 이 실습 데이터에서는 개발본부, 데이터전략팀, 영업본부, 인사총무팀에 위쪽 이상치 후보가 있고, 고객운영팀에는 IQR 기준 이상치가 없다.

### 4단계. 결과 해석하기

아래 문장을 참고해 자기 말로 해석한다.

```text
개발본부는 평균과 중앙값이 가장 높다. 고객운영팀은 평균과 중앙값이 가장 낮다. 데이터전략팀과 인사총무팀은 범위가 47,500,000원으로 넓고, 박스플롯에서도 연봉이 넓게 퍼진 모습을 확인할 수 있다. 개발본부에는 위쪽 이상치 후보가 6건 있다.
```

## 직접 해보기. 평균 왜곡 사례 찾기

### 미션

`cafe_sales_clean.xlsx`의 `sales` 시트에서 상품별 주문금액 `amount`를 비교한다. 평균이 중앙값보다 많이 높은 상품을 찾아, 평균만 보면 왜 보통 주문금액을 잘못 이해할 수 있는지 설명한다.

### 조건

1. `cafe_sales_clean.xlsx`를 연다.
2. 새 시트를 만들고 이름을 `item_amount_summary`로 바꾼다.
3. 상품별로 주문 건수, 평균, 중앙값, 최빈값, 최솟값, 최댓값, 범위, 표준편차를 계산한다.
4. `mean_minus_median`을 만든다.
5. `mean_minus_median`이 가장 큰 상품을 찾는다.
6. 그 상품의 `qty` 분포를 확인해 왜 평균이 올라갔는지 설명한다.

### 계산표 만들기

A1:K1에 아래 제목을 입력한다.

| 셀 | 제목 |
|---|---|
| A1 | `item` |
| B1 | `count` |
| C1 | `mean` |
| D1 | `median` |
| E1 | `mode` |
| F1 | `min` |
| G1 | `max` |
| H1 | `range` |
| I1 | `std_dev` |
| J1 | `mean_minus_median` |
| K1 | `mean_to_median_ratio` |

A2 셀에 입력한다.

```excel
=SORT(UNIQUE(sales!$G$2:$G$10425))
```

`SORT`와 `UNIQUE` 함수가 없는 엑셀을 사용하는 경우에는 다음 방식으로 상품 목록을 만든다.

1. `sales` 시트의 `item` 열을 빈 보조열에 복사한다.
2. 보조열에서 중복된 항목을 제거해 고유 상품 목록만 남긴다.
3. 남은 상품 목록을 오름차순으로 정렬한 뒤 `item_amount_summary` 시트 A2부터 붙여넣는다.
4. 또는 `sales` 시트를 `item` 기준으로 정렬한 뒤 상품명이 바뀌는 첫 행의 상품명만 A2부터 복사한다.

B2 셀에 입력한다.

```excel
=COUNTIF(sales!$G$2:$G$10425,A2)
```

C2 셀에 입력한다.

```excel
=AVERAGEIF(sales!$G$2:$G$10425,A2,sales!$J$2:$J$10425)
```

D2 셀에 입력한다.

```excel
=MEDIAN(FILTER(sales!$J$2:$J$10425,sales!$G$2:$G$10425=A2))
```

E2 셀에 입력한다.

```excel
=MODE.SNGL(FILTER(sales!$J$2:$J$10425,sales!$G$2:$G$10425=A2))
```

F2 셀에 입력한다.

```excel
=MINIFS(sales!$J$2:$J$10425,sales!$G$2:$G$10425,A2)
```

G2 셀에 입력한다.

```excel
=MAXIFS(sales!$J$2:$J$10425,sales!$G$2:$G$10425,A2)
```

H2 셀에 입력한다.

```excel
=G2-F2
```

I2 셀에 입력한다.

```excel
=STDEV.S(FILTER(sales!$J$2:$J$10425,sales!$G$2:$G$10425=A2))
```

J2 셀에 입력한다.

```excel
=C2-D2
```

K2 셀에 입력한다.

```excel
=C2/D2
```

B2:K2를 상품 목록 마지막 행까지 채운다. 그 다음 J열을 기준으로 내림차순 정렬한다.

`FILTER` 함수가 없는 엑셀에서는 `sales` 시트를 `item` 기준으로 정렬해 상품별 `amount` 범위를 직접 참조한다. 또는 `item` 필터로 한 상품만 보이게 한 뒤 보이는 `amount` 값을 보조열에 복사하고, 그 범위로 `MEDIAN`, `MODE.SNGL`, `STDEV.S`를 계산한다.
