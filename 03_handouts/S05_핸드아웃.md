# S05 핸드아웃: 중심·산포·분포·박스플롯(상자 수염 그림)

## 사용할 데이터

따라하기에서는 수업 홈의 데이터 받기 페이지에서 내려받은 직원 연봉 파일을 사용합니다. 엑셀 파일 형식으로 내려받아도 괜찮습니다. 이 파일로 부서별 연봉이 어디쯤 모여 있고, 얼마나 넓게 퍼져 있는지 확인해 봅니다.

- 시트 이름: `employees`
- 데이터 규모: 1,200행 × 9열
- 사용할 컬럼: `dept`, `salary`

직접 해보기에서는 수업 홈의 데이터 받기 페이지에서 내려받은 카페 주문 파일을 사용합니다. 이 파일로 상품별 주문금액을 비교하면서 평균이 보통의 느낌을 얼마나 잘 설명하는지 확인해 봅니다.

- 시트 이름: `sales`
- 데이터 규모: 10,424행 × 12열
- 사용할 컬럼: `item`, `qty`, `amount`

## 오늘의 목표

직원 연봉 데이터로 부서별 분포를 비교합니다. 여기서 분포란 숫자들이 낮은 곳부터 높은 곳까지 어떻게 놓여 있는지를 말합니다. 키를 잴 때 반에서 가장 작은 학생, 가운데쯤인 학생, 가장 큰 학생을 함께 보는 것과 비슷합니다.

평균만 구하고 끝내지 않습니다. 중앙값, 최빈값, 범위, 표준편차까지 함께 구한 뒤 박스플롯으로 연봉의 퍼짐을 확인합니다. 마지막에는 카페 주문 데이터에서 평균이 보통 주문금액을 과장해 보이게 만드는 사례를 찾아봅니다.

## 따라하기 1. 부서별 연봉 요약표 만들기

### 1단계. 데이터 받기 페이지에서 직원 연봉 파일 내려받고 컬럼 확인하기

이 단계는 계산을 시작하기 전에 우리가 어떤 열을 보고 있는지 맞추기 위해 필요합니다. 엑셀에서 열을 잘못 잡으면 공식은 맞아도 결과가 전혀 다른 숫자로 나올 수 있습니다.

1. 수업 홈의 데이터 받기 페이지에서 직원 연봉 파일을 내려받아 엽니다.
2. `employees` 시트를 확인합니다.
3. 다음 컬럼을 찾습니다.

| 컬럼 | 의미 |
|---|---|
| `dept` | 부서 |
| `salary` | 연봉 |

이 파일은 1행이 제목이고, 실제 데이터는 2행부터 1,201행까지 있습니다. `dept`는 B열, `salary`는 G열입니다. `dept`는 “이 직원이 어느 부서에 속해 있는가”를 알려 주고, `salary`는 “그 직원의 연봉이 얼마인가”를 알려 줍니다.

### 2단계. 요약 시트 만들기

이 단계는 부서별 계산 결과를 한곳에 모아 보기 위해 필요합니다. 원본 데이터는 직원 한 명당 한 줄이지만, 요약 시트에서는 부서 하나당 한 줄로 정리합니다.

1. 새 시트를 만듭니다.
2. 시트 이름을 `dept_salary_summary`로 바꿉니다.
3. A1:O1에 아래 제목을 입력합니다.

| 셀 | 제목 |
|---|---|
| A1 | `dept` |
| B1 | `count` |
| C1 | `mean` |
| D1 | `median` |
| E1 | `mode` |
| F1 | `min` |
| G1 | `max` |
| H1 | `range` |
| I1 | `std_dev` |
| J1 | `q1` |
| K1 | `q3` |
| L1 | `iqr` |
| M1 | `lower_fence` |
| N1 | `upper_fence` |
| O1 | `outlier_count` |

4. A2:A6에 부서명을 입력합니다.

```text
개발본부
데이터전략팀
영업본부
인사총무팀
고객운영팀
```

여기서 A열의 부서명은 사전에서 단어를 찾을 때 쓰는 검색어처럼 작동합니다. 뒤에서 엑셀 함수가 A2의 `개발본부`를 보고, 원본 데이터에서 개발본부 직원만 찾아 계산하게 됩니다.

### 3단계. 직원 수와 중심값 계산하기

이 단계는 각 부서의 “대표적인 연봉 위치”를 보기 위해 필요합니다. 평균은 전체 수준을 빠르게 보여 주고, 중앙값은 줄을 세웠을 때 가운데 값을 보여 주며, 최빈값은 가장 자주 나오는 값을 보여 줍니다.

B2 셀에 입력합니다.

```excel
=COUNTIF(employees!$B$2:$B$1201,A2)
```

C2 셀에 입력합니다.

```excel
=AVERAGEIF(employees!$B$2:$B$1201,A2,employees!$G$2:$G$1201)
```

D2 셀에 입력합니다.

```excel
=MEDIAN(FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A2))
```

E2 셀에 입력합니다.

```excel
=MODE.SNGL(FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A2))
```

B2:E2를 6행까지 채웁니다.

뜻은 이렇게 보면 됩니다.

- `count`: 부서별 직원 수입니다. 계산할 사람이 몇 명인지 세는 값입니다.
- `mean`: 부서별 평균 연봉입니다. 부서 전체 연봉을 직원 수로 나눈 값입니다.
- `median`: 부서별 중앙 연봉입니다. 연봉을 낮은 순서부터 높은 순서로 줄 세웠을 때 가운데에 있는 값입니다.
- `mode`: 부서별 가장 자주 나오는 연봉입니다. 같은 연봉이 여러 번 반복될 때 특히 의미가 있습니다.

평균과 중앙값을 함께 보는 이유가 있습니다. 아주 높은 연봉 몇 명이 있으면 평균은 위로 끌려 올라갈 수 있습니다. 중앙값은 가운데 위치를 보기 때문에 그런 큰 값의 영향을 평균보다 덜 받습니다.

### 4단계. 범위와 표준편차 계산하기

이 단계는 연봉이 얼마나 넓게 퍼져 있는지 보기 위해 필요합니다. 평균이 비슷한 두 부서라도 한 부서는 연봉이 고르게 모여 있고, 다른 부서는 낮은 연봉과 높은 연봉이 크게 섞여 있을 수 있습니다.

F2 셀에 입력합니다.

```excel
=MINIFS(employees!$G$2:$G$1201,employees!$B$2:$B$1201,A2)
```

G2 셀에 입력합니다.

```excel
=MAXIFS(employees!$G$2:$G$1201,employees!$B$2:$B$1201,A2)
```

H2 셀에 입력합니다.

```excel
=G2-F2
```

I2 셀에 입력합니다.

```excel
=STDEV.S(FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A2))
```

F2:I2를 6행까지 채웁니다.

뜻은 이렇게 보시면 됩니다.

- `min`: 최저 연봉입니다.
- `max`: 최고 연봉입니다.
- `range`: 최고 연봉과 최저 연봉의 차이입니다.
- `std_dev`: 평균 주변에서 값이 흩어진 정도입니다. 표준편차라고 읽고, 값들이 평균 근처에 모여 있는지 넓게 퍼져 있는지를 보는 숫자입니다.

범위는 자로 양끝을 재는 것처럼 가장 낮은 값과 가장 높은 값의 차이를 봅니다. 표준편차는 전체 값들이 평균에서 얼마나 떨어져 있는지를 함께 봅니다.

### 5단계. 사분위수와 이상치 기준 계산하기

이 단계는 박스플롯을 해석할 기준을 만들기 위해 필요합니다. 사분위수는 데이터를 낮은 순서부터 세웠을 때 25%, 50%, 75% 지점을 나누는 값입니다.

J2 셀에 입력합니다.

```excel
=QUARTILE.INC(FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A2),1)
```

K2 셀에 입력합니다.

```excel
=QUARTILE.INC(FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A2),3)
```

L2 셀에 입력합니다.

```excel
=K2-J2
```

M2 셀에 입력합니다.

```excel
=J2-1.5*L2
```

N2 셀에 입력합니다.

```excel
=K2+1.5*L2
```

O2 셀에 입력합니다.

```excel
=COUNTIFS(employees!$B$2:$B$1201,A2,employees!$G$2:$G$1201,"<"&M2)+COUNTIFS(employees!$B$2:$B$1201,A2,employees!$G$2:$G$1201,">"&N2)
```

J2:O2를 6행까지 채웁니다.

`FILTER` 함수가 없는 엑셀을 사용하는 경우에는 중앙값, 최빈값, 표준편차, 사분위수 계산을 다음 방식으로 대체합니다.

1. `employees` 시트의 `dept`, `salary` 열을 빈 보조 영역에 복사합니다.
2. 보조 영역을 `dept` 기준으로 정렬해 같은 부서의 `salary`가 연속되게 만듭니다. 또는 `dept` 필터로 한 부서만 보이게 한 뒤 보이는 `salary` 값을 보조열에 복사합니다.
3. 각 부서의 `salary` 범위를 직접 참조해 `=MEDIAN(범위)`, `=MODE.SNGL(범위)`, `=STDEV.S(범위)`, `=QUARTILE.INC(범위,1)`, `=QUARTILE.INC(범위,3)`을 입력합니다.
4. 예를 들어 개발본부 `salary`가 보조열 Q2:Q371에 있으면 D2에는 `=MEDIAN($Q$2:$Q$371)`, E2에는 `=MODE.SNGL($Q$2:$Q$371)`, I2에는 `=STDEV.S($Q$2:$Q$371)`, J2에는 `=QUARTILE.INC($Q$2:$Q$371,1)`, K2에는 `=QUARTILE.INC($Q$2:$Q$371,3)`을 입력합니다.

이상치 기준은 입문 단계에서 이렇게 이해하면 됩니다. 이상치는 “무조건 틀린 값”이라는 뜻이 아니라, 다른 값들과 비교했을 때 유난히 멀리 떨어져 있어서 따로 살펴볼 만한 값입니다.

```text
IQR = Q3 - Q1
아래쪽 이상치 후보 = Q1 - 1.5 × IQR 보다 작은 값
위쪽 이상치 후보 = Q3 + 1.5 × IQR 보다 큰 값
```

### 6단계. 검산값 확인하기

이 단계는 내가 만든 요약표가 크게 틀리지 않았는지 확인하기 위해 필요합니다. 계산 결과가 아래 값과 비슷한 흐름이면, 수식 범위와 조건을 제대로 잡은 것입니다.

아래 값과 큰 흐름이 맞으면 요약표가 정상적으로 만들어진 것입니다. 평균과 표준편차는 소수 없이 반올림해 확인합니다.

| 부서 | 직원 수 | 평균 | 중앙값 | 범위 | 표준편차 | 이상치 수 |
|---|---:|---:|---:|---:|---:|---:|
| 개발본부 | 370 | 74,190,541 | 73,950,000 | 42,100,000 | 8,094,278 | 6 |
| 데이터전략팀 | 250 | 68,902,000 | 68,450,000 | 47,500,000 | 8,955,824 | 2 |
| 영업본부 | 240 | 63,087,500 | 62,700,000 | 44,900,000 | 8,070,936 | 3 |
| 인사총무팀 | 180 | 59,414,444 | 59,100,000 | 47,500,000 | 8,771,463 | 3 |
| 고객운영팀 | 160 | 53,445,625 | 53,300,000 | 36,900,000 | 8,456,056 | 0 |

여기서 개발본부 평균 연봉은 74,190,541이고, 고객운영팀 평균 연봉은 53,445,625입니다. 평균만 보면 개발본부가 가장 높고 고객운영팀이 가장 낮다는 점이 보입니다. 하지만 범위, 표준편차, 이상치 수까지 같이 봐야 “얼마나 고르게 분포했는지”까지 말할 수 있습니다.

## 따라하기 2. 부서별 연봉 박스플롯 만들기

### 1단계. 박스플롯용 시트 만들기

이 단계는 차트에 넣을 데이터를 차트가 읽기 쉬운 모양으로 바꾸기 위해 필요합니다. 박스플롯은 부서별 연봉 목록을 각각 한 열에 두면 만들기 쉽습니다.

1. 새 시트를 만듭니다.
2. 시트 이름을 `dept_boxplot`으로 바꿉니다.
3. A1:E1에 부서명을 입력합니다.

| 셀 | 제목 |
|---|---|
| A1 | `개발본부` |
| B1 | `데이터전략팀` |
| C1 | `영업본부` |
| D1 | `인사총무팀` |
| E1 | `고객운영팀` |

### 2단계. 부서별 연봉을 열로 펼치기

이 단계는 원본 데이터에 섞여 있는 연봉을 부서별로 따로 모으기 위해 필요합니다. 사전에서 같은 단어가 들어간 항목만 골라 새 목록을 만드는 느낌으로 보시면 됩니다.

A2 셀에 입력합니다.

```excel
=FILTER(employees!$G$2:$G$1201,employees!$B$2:$B$1201=A$1)
```

A2 셀의 수식을 E2까지 복사합니다.

각 부서의 연봉이 아래쪽으로 자동으로 펼쳐집니다. 부서마다 직원 수가 다르므로 열마다 채워지는 길이가 다를 수 있습니다.

`FILTER` 함수가 없는 엑셀을 사용하는 경우에는 다음 방식으로 대체합니다.

1. `employees` 시트에서 `dept` 열에 필터를 겁니다.
2. `개발본부`만 선택합니다.
3. 보이는 `salary` 값을 복사해 `dept_boxplot` 시트 A열에 붙여넣습니다.
4. 나머지 부서도 B열부터 E열까지 같은 방식으로 붙여넣습니다.

### 3단계. 박스플롯 삽입하기

이 단계는 숫자 목록을 그림으로 바꿔 한눈에 비교하기 위해 필요합니다. 표에서는 여러 값을 하나씩 읽어야 하지만, 박스플롯에서는 가운데 위치와 퍼짐, 이상치 후보가 한 화면에 보입니다.

1. `dept_boxplot` 시트에서 A1:E371 범위를 선택합니다.
2. 상단 메뉴에서 `삽입`을 클릭합니다.
3. `통계 차트 삽입`을 클릭합니다.
4. `박스플롯`을 선택합니다.
5. 차트 제목을 `부서별 연봉 분포`로 바꿉니다.

차트에서 따로 찍힌 점은 이상치 후보로 봅니다. 이 실습 데이터에서는 개발본부, 데이터전략팀, 영업본부, 인사총무팀에 위쪽 이상치 후보가 있고, 고객운영팀에는 IQR 기준 이상치가 없습니다.

### 4단계. 결과 해석하기

이 단계는 만든 표와 차트를 말로 설명하는 연습을 하기 위해 필요합니다. 데이터 분석에서는 계산을 끝내는 것보다, 그 숫자가 무엇을 뜻하는지 설명하는 일이 더 중요합니다.

아래 문장을 참고해 자기 말로 해석해 보세요.

```text
개발본부는 평균과 중앙값이 가장 높다. 고객운영팀은 평균과 중앙값이 가장 낮다. 데이터전략팀과 인사총무팀은 범위가 47,500,000원으로 넓고, 박스플롯에서도 연봉이 넓게 퍼진 모습을 확인할 수 있다. 개발본부에는 위쪽 이상치 후보가 6건 있다.
```

여기서 봐야 할 점은 “어느 부서가 높다”에서 멈추지 않는 것입니다. 평균과 중앙값이 높은지, 범위가 넓은지, 박스플롯에서 점이 따로 찍히는지까지 함께 말해 보세요.

## 직접 해보기. 평균 왜곡 사례 찾기

### 미션

수업 홈의 데이터 받기 페이지에서 내려받은 카페 주문 파일의 `sales` 시트에서 상품별 주문금액 `amount`를 비교합니다. 평균이 중앙값보다 많이 높은 상품을 찾아, 평균만 보면 왜 보통 주문금액을 잘못 이해할 수 있는지 설명해 보세요.

평균이 중앙값보다 높다는 말은 큰 주문금액 몇 건이 평균을 끌어올렸을 가능성이 있다는 뜻입니다. 예를 들어 보통은 한 개씩 주문하는 상품인데, 어떤 손님들이 여러 개를 한 번에 주문했다면 평균 주문금액은 올라갈 수 있습니다.

### 조건

1. 수업 홈의 데이터 받기 페이지에서 카페 주문 파일을 내려받아 엽니다.
2. 새 시트를 만들고 이름을 `item_amount_summary`로 바꿉니다.
3. 상품별로 주문 건수, 평균, 중앙값, 최빈값, 최솟값, 최댓값, 범위, 표준편차를 계산합니다.
4. `mean_minus_median`을 만듭니다.
5. `mean_minus_median`이 가장 큰 상품을 찾습니다.
6. 그 상품의 `qty` 분포를 확인해 왜 평균이 올라갔는지 설명합니다.

`qty`는 주문 수량입니다. 같은 치즈케이크라도 1개 주문과 여러 개 주문은 `amount`가 달라지므로, 평균이 왜 올라갔는지 볼 때 `qty`를 함께 확인해야 합니다.

### 계산표 만들기

A1:K1에 아래 제목을 입력합니다.

| 셀 | 제목 |
|---|---|
| A1 | `item` |
| B1 | `count` |
| C1 | `mean` |
| D1 | `median` |
| E1 | `mode` |
| F1 | `min` |
| G1 | `max` |
| H1 | `range` |
| I1 | `std_dev` |
| J1 | `mean_minus_median` |
| K1 | `mean_to_median_ratio` |

A2 셀에 입력합니다.

```excel
=SORT(UNIQUE(sales!$G$2:$G$10425))
```

`SORT`와 `UNIQUE` 함수가 없는 엑셀을 사용하는 경우에는 다음 방식으로 상품 목록을 만듭니다.

1. `sales` 시트의 `item` 열을 빈 보조열에 복사합니다.
2. 보조열에서 중복된 항목을 제거해 고유 상품 목록만 남깁니다.
3. 남은 상품 목록을 오름차순으로 정렬한 뒤 `item_amount_summary` 시트 A2부터 붙여넣습니다.
4. 또는 `sales` 시트를 `item` 기준으로 정렬한 뒤 상품명이 바뀌는 첫 행의 상품명만 A2부터 복사합니다.

B2 셀에 입력합니다.

```excel
=COUNTIF(sales!$G$2:$G$10425,A2)
```

C2 셀에 입력합니다.

```excel
=AVERAGEIF(sales!$G$2:$G$10425,A2,sales!$J$2:$J$10425)
```

D2 셀에 입력합니다.

```excel
=MEDIAN(FILTER(sales!$J$2:$J$10425,sales!$G$2:$G$10425=A2))
```

E2 셀에 입력합니다.

```excel
=MODE.SNGL(FILTER(sales!$J$2:$J$10425,sales!$G$2:$G$10425=A2))
```

F2 셀에 입력합니다.

```excel
=MINIFS(sales!$J$2:$J$10425,sales!$G$2:$G$10425,A2)
```

G2 셀에 입력합니다.

```excel
=MAXIFS(sales!$J$2:$J$10425,sales!$G$2:$G$10425,A2)
```

H2 셀에 입력합니다.

```excel
=G2-F2
```

I2 셀에 입력합니다.

```excel
=STDEV.S(FILTER(sales!$J$2:$J$10425,sales!$G$2:$G$10425=A2))
```

J2 셀에 입력합니다.

```excel
=C2-D2
```

K2 셀에 입력합니다.

```excel
=C2/D2
```

B2:K2를 상품 목록 마지막 행까지 채웁니다. 그 다음 J열을 기준으로 내림차순 정렬합니다.

`mean_minus_median`은 평균에서 중앙값을 뺀 값입니다. 이 값이 클수록 평균이 중앙값보다 많이 높다는 뜻입니다. `mean_to_median_ratio`는 평균이 중앙값의 몇 배인지 보는 값입니다.

`FILTER` 함수가 없는 엑셀에서는 `sales` 시트를 `item` 기준으로 정렬해 상품별 `amount` 범위를 직접 참조합니다. 또는 `item` 필터로 한 상품만 보이게 한 뒤 보이는 `amount` 값을 보조열에 복사하고, 그 범위로 `MEDIAN`, `MODE.SNGL`, `STDEV.S`를 계산합니다.

### 제출할 답

아래 형식으로 한 문장을 작성합니다.

```text
평균 왜곡 사례로 볼 수 있는 상품은 ______이다. 이 상품의 평균 주문금액은 ______원이고 중앙값은 ______원이다. 평균이 더 큰 이유는 ______ 때문이다. 따라서 보통 주문금액을 말할 때는 평균만 말하기보다 중앙값과 최빈값도 함께 봐야 한다.
```

문장을 쓸 때는 “평균이 높다”에서 끝내지 말고, 왜 높아졌는지를 `qty`와 연결해 보세요. 보통 주문은 한 개 주문에 가까운데 여러 개 주문이 섞여 평균을 올렸다면, 그 점을 말로 풀어 주면 됩니다.
