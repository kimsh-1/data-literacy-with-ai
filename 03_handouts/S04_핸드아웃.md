# S04 핸드아웃: VLOOKUP/XLOOKUP과 조건부 집계

## 실습 목표

이번 실습에서는 주문 데이터에 상품 단가와 고객의 지역, 등급을 붙이고 주문별 매출을 계산한다. 끝으로 지역별 매출을 구한 뒤, 직접 해보기 미션에서 등급별 객단가까지 계산한다.

## 사용할 파일

```text
/mnt/d/ai-data-analysis-course/02_datasets/shop_orders/shop_orders_clean.xlsx
```

파일을 열면 세 개의 시트가 있다.

- `orders`: 주문 목록, 20,000행
- `customers`: 고객 정보, 3,000행
- `products`: 상품 정보, 200행

## 따라하기

### 1단계. 세 시트의 역할 확인하기

1. `shop_orders_clean.xlsx`를 연다.
2. 아래 시트를 차례로 클릭한다.
   - `orders`
   - `customers`
   - `products`
3. 각 시트에서 다음 컬럼을 확인한다.

| 시트 | 확인할 컬럼 | 의미 |
|---|---|---|
| `orders` | `customer_id` | 고객을 찾는 번호 |
| `orders` | `product_id` | 상품을 찾는 번호 |
| `orders` | `qty` | 주문 수량 |
| `orders` | `discount_rate` | 할인율 |
| `customers` | `customer_id`, `region`, `grade` | 고객번호별 지역과 등급 |
| `products` | `product_id`, `price` | 상품번호별 단가 |

`orders`에는 매출이 없다. 매출을 구하려면 `products`에서 단가부터 가져와야 한다.

### 2단계. orders 시트에 새 열 만들기

`orders` 시트에서 F열 오른쪽에 다음 제목을 추가한다.

| 열 | 제목 |
|---|---|
| G열 | `product_name` |
| H열 | `category` |
| I열 | `price` |
| J열 | `region` |
| K열 | `grade` |
| L열 | `sales` |

### 3단계. VLOOKUP으로 상품 정보 가져오기

`orders` 시트의 G2 셀에 입력한다.

```excel
=VLOOKUP(C2,products!$A:$E,2,FALSE)
```

H2 셀에 입력한다.

```excel
=VLOOKUP(C2,products!$A:$E,3,FALSE)
```

I2 셀에 입력한다.

```excel
=VLOOKUP(C2,products!$A:$E,4,FALSE)
```

각 공식의 뜻은 다음과 같다.

- `C2`: 현재 주문의 상품번호
- `products!$A:$E`: 상품번호를 찾을 상품 표
- `2`, `3`, `4`: 상품명, 카테고리, 단가가 있는 열 번호
- `FALSE`: 정확히 일치하는 상품번호만 찾기

G2부터 I2까지 선택한 뒤 20,001행까지 채운다.

### 4단계. VLOOKUP으로 고객 정보 가져오기

J2 셀에 입력한다.

```excel
=VLOOKUP(B2,customers!$A:$G,5,FALSE)
```

K2 셀에 입력한다.

```excel
=VLOOKUP(B2,customers!$A:$G,7,FALSE)
```

뜻은 다음과 같다.

- `B2`: 현재 주문의 고객번호
- `customers!$A:$G`: 고객번호를 찾을 고객 표
- `5`: `region` 열
- `7`: `grade` 열
- `FALSE`: 정확히 일치하는 고객번호만 찾기

J2부터 K2까지 선택한 뒤 20,001행까지 채운다.

### 5단계. XLOOKUP 방식도 비교하기

엑셀에 `XLOOKUP`이 있다면 같은 작업을 아래처럼 쓸 수도 있다. 앞서 VLOOKUP으로 값을 이미 채웠다면 다시 입력할 필요는 없고, 두 함수의 구조를 견줘 보는 셈 치고 읽으면 된다.

단가를 가져오는 공식:

```excel
=XLOOKUP(C2,products!$A:$A,products!$D:$D,"확인필요")
```

지역을 가져오는 공식:

```excel
=XLOOKUP(B2,customers!$A:$A,customers!$E:$E,"확인필요")
```

등급을 가져오는 공식:

```excel
=XLOOKUP(B2,customers!$A:$A,customers!$G:$G,"확인필요")
```

`VLOOKUP`은 가져올 열을 숫자로 세고, `XLOOKUP`은 찾을 열과 가져올 열을 직접 지정한다.

### 6단계. 주문별 매출 계산하기

L2 셀에 입력한다.

```excel
=I2*E2*(1-F2)
```

뜻은 다음과 같다.

```text
매출 = 단가 × 수량 × (1 - 할인율)
```

L2 셀을 20,001행까지 채운다.

확인할 값:

- L2: `22900`
- L3: `35910`
- L4: `105800`

### 7단계. 지역별 매출 집계표 만들기

1. 새 시트를 만든다.
2. 시트 이름을 `region_summary`로 바꾼다.
3. A1:D1에 제목을 입력한다.

| 셀 | 제목 |
|---|---|
| A1 | `region` |
| B1 | `order_count` |
| C1 | `total_sales` |
| D1 | `avg_order_value` |

4. `orders` 시트의 J2:J20001을 복사해 `region_summary`의 A2에 붙여넣는다.
5. A열을 선택하고 데이터 탭에서 중복 제거를 실행한다.
6. A열을 오름차순으로 정렬한다.

B2 셀에 입력한다.

```excel
=COUNTIF(orders!$J$2:$J$20001,A2)
```

C2 셀에 입력한다.

```excel
=SUMIF(orders!$J$2:$J$20001,A2,orders!$L$2:$L$20001)
```

D2 셀에 입력한다.

```excel
=C2/B2
```

B2:D2를 마지막 지역 행까지 채운다.

검산 기준:

- `order_count` 합계: `20000`
- `total_sales` 합계: `1938489803`
- 서울특별시 매출: `570482124`
- 경기도 매출: `530726511`

## 직접 해보기

### 미션. 등급별 객단가 구하기

고객 등급별로 주문 건수, 매출 합계, 객단가를 구한다.

1. 새 시트를 만든다.
2. 시트 이름을 `grade_summary`로 바꾼다.
3. A1:D1에 제목을 입력한다.

| 셀 | 제목 |
|---|---|
| A1 | `grade` |
| B1 | `order_count` |
| C1 | `total_sales` |
| D1 | `avg_order_value` |

4. A2:A5에 등급을 입력한다.

```text
VIP
골드
실버
브론즈
```

5. B2:D2에 알맞은 `COUNTIF`, `SUMIF`, 나눗셈 공식을 작성한다.
6. B2:D2를 5행까지 채운다.
7. 어떤 등급의 객단가가 가장 높은지 한 문장으로 적는다.

힌트:

- 등급은 `orders` 시트의 K열에 있다.
- 매출은 `orders` 시트의 L열에 있다.
- 객단가는 매출 합계를 주문 건수로 나눈 값이다.
