# S04 핸드아웃: VLOOKUP/XLOOKUP과 조건부 집계

## 실습 목표

주문 데이터에는 고객번호와 상품번호는 있지만, 고객의 지역, 등급, 상품 단가는 아직 붙어 있지 않습니다. 이 실습에서는 주문 표에 상품 정보와 고객 정보를 찾아 붙이고, 주문별 매출을 계산해 봅니다. 마지막에는 지역별 매출을 구한 뒤, 직접 해보기 미션에서 등급별 객단가까지 계산합니다.

엑셀에서는 이런 작업을 “표끼리 연결한다”고 볼 수 있습니다. 주문 표가 영수증이라면, 고객 표와 상품 표는 사전입니다. 번호를 보고 사전에서 뜻을 찾듯이, 고객번호와 상품번호를 기준으로 필요한 정보를 가져오는 연습을 해 봅니다.

## 사용할 파일

수업 홈의 데이터 받기 페이지에서 쇼핑몰 주문 파일을 내려받아 여세요. 엑셀 파일을 받아도 되고, 엑셀로도 열리는 표 파일인 CSV로 받아도 됩니다. 이 실습에서는 여러 시트가 함께 들어 있는 엑셀 파일을 기준으로 설명합니다.

파일을 열면 세 개의 시트가 있습니다.

- `orders`: 주문 목록, 20,000행
- `customers`: 고객 정보, 3,000행
- `products`: 상품 정보, 200행

## 따라하기

### 1단계. 세 시트의 역할 확인하기

이 단계에서는 어떤 표에서 무엇을 가져와야 하는지 먼저 확인합니다. 재료가 어디에 있는지 알아야 공식을 쓸 때 길을 잃지 않습니다.

1. 데이터 받기 페이지에서 내려받은 쇼핑몰 주문 파일을 엽니다.
2. 아래 시트를 차례로 클릭해 보세요.
   - `orders`
   - `customers`
   - `products`
3. 각 시트에서 다음 컬럼을 확인합니다.

| 시트 | 확인할 컬럼 | 의미 |
|---|---|---|
| `orders` | `customer_id` | 고객을 찾는 번호 |
| `orders` | `product_id` | 상품을 찾는 번호 |
| `orders` | `qty` | 주문 수량 |
| `orders` | `discount_rate` | 할인율 |
| `customers` | `customer_id`, `region`, `grade` | 고객번호별 지역과 등급 |
| `products` | `product_id`, `price` | 상품번호별 단가 |

`orders`에는 매출이 없습니다. 주문 수량과 할인율은 있지만, 상품 단가가 없기 때문입니다. 매출을 구하려면 먼저 `products`에서 단가를 가져와야 합니다.

### 2단계. orders 시트에 새 열 만들기

이 단계에서는 가져온 값과 계산 결과를 담을 자리를 만듭니다. 빈 칸을 먼저 만들어 두면, 어떤 정보를 붙이고 있는지 눈으로 따라가기 쉽습니다.

`orders` 시트에서 F열 오른쪽에 다음 제목을 추가합니다.

| 열 | 제목 |
|---|---|
| G열 | `product_name` |
| H열 | `category` |
| I열 | `price` |
| J열 | `region` |
| K열 | `grade` |
| L열 | `sales` |

G열부터 I열까지는 상품 표에서 가져올 정보입니다. J열과 K열은 고객 표에서 가져올 정보입니다. L열은 단가, 수량, 할인율을 이용해 직접 계산할 매출입니다.

### 3단계. VLOOKUP으로 상품 정보 가져오기

이 단계에서는 상품번호를 기준으로 상품명, 카테고리, 단가를 가져옵니다. 사전에서 단어를 찾듯이, `product_id`를 보고 `products` 시트에서 필요한 값을 찾는 작업입니다.

`orders` 시트의 G2 셀에 입력합니다.

```excel
=VLOOKUP(C2,products!$A:$E,2,FALSE)
```

H2 셀에 입력합니다.

```excel
=VLOOKUP(C2,products!$A:$E,3,FALSE)
```

I2 셀에 입력합니다.

```excel
=VLOOKUP(C2,products!$A:$E,4,FALSE)
```

각 공식의 부분은 이렇게 읽습니다.

- `C2`: 현재 주문의 상품번호입니다.
- `products!$A:$E`: 상품번호를 찾을 상품 표입니다.
- `2`, `3`, `4`: 상품명, 카테고리, 단가가 있는 열 번호입니다.
- `FALSE`: 비슷한 상품번호가 아니라 정확히 같은 상품번호만 찾겠다는 뜻입니다.

여기서 `FALSE`는 꼭 확인해 주세요. 상품번호는 대충 비슷한 값을 찾으면 안 됩니다. 주민등록번호나 학번처럼 정확히 맞는 번호를 찾아야 합니다.

G2부터 I2까지 선택한 뒤 20,001행까지 채웁니다. 첫 주문에만 공식을 쓰는 것이 아니라, 20,000건의 주문 전체에 같은 방식으로 상품 정보를 붙이는 과정입니다.

### 4단계. VLOOKUP으로 고객 정보 가져오기

이 단계에서는 고객번호를 기준으로 지역과 등급을 가져옵니다. 상품번호로 상품 사전을 찾았던 것처럼, 이번에는 고객번호로 고객 사전을 찾는다고 보면 됩니다.

J2 셀에 입력합니다.

```excel
=VLOOKUP(B2,customers!$A:$G,5,FALSE)
```

K2 셀에 입력합니다.

```excel
=VLOOKUP(B2,customers!$A:$G,7,FALSE)
```

각 부분은 이렇게 읽습니다.

- `B2`: 현재 주문의 고객번호입니다.
- `customers!$A:$G`: 고객번호를 찾을 고객 표입니다.
- `5`: `region` 열입니다.
- `7`: `grade` 열입니다.
- `FALSE`: 정확히 일치하는 고객번호만 찾겠다는 뜻입니다.

J2부터 K2까지 선택한 뒤 20,001행까지 채웁니다. 이렇게 하면 주문 한 줄마다 고객의 지역과 등급이 붙습니다. 이제 주문 표만 보아도 “어느 지역의 어떤 등급 고객이 무엇을 샀는지”를 볼 수 있습니다.

### 5단계. XLOOKUP 방식도 비교하기

이 단계에서는 같은 찾기 작업을 `XLOOKUP`으로도 읽어 봅니다. 이미 `VLOOKUP`으로 값을 채웠다면 다시 입력하지 않아도 됩니다. 두 함수가 어떻게 다른지 구조를 비교하는 것이 목적입니다.

엑셀에 `XLOOKUP`이 있다면 같은 작업을 아래처럼 쓸 수도 있습니다. 앞서 VLOOKUP으로 값을 이미 채웠다면 다시 입력할 필요는 없고, 두 함수의 구조를 견줘 보는 셈 치고 읽으면 됩니다.

단가를 가져오는 공식:

```excel
=XLOOKUP(C2,products!$A:$A,products!$D:$D,"확인필요")
```

지역을 가져오는 공식:

```excel
=XLOOKUP(B2,customers!$A:$A,customers!$E:$E,"확인필요")
```

등급을 가져오는 공식:

```excel
=XLOOKUP(B2,customers!$A:$A,customers!$G:$G,"확인필요")
```

`VLOOKUP`은 가져올 열을 숫자로 셉니다. 그래서 상품 표에서 단가가 몇 번째 열인지 직접 세어야 합니다. `XLOOKUP`은 찾을 열과 가져올 열을 직접 지정합니다. 사전의 왼쪽 페이지에서 단어를 찾고, 오른쪽 페이지의 뜻을 바로 가져오는 느낌에 가깝습니다.

### 6단계. 주문별 매출 계산하기

이 단계에서는 주문 한 건마다 실제 매출을 계산합니다. 단가만으로는 매출이 아닙니다. 몇 개를 샀는지, 할인이 있었는지까지 반영해야 합니다.

L2 셀에 입력합니다.

```excel
=I2*E2*(1-F2)
```

뜻은 이렇게 읽습니다.

```text
매출 = 단가 × 수량 × (1 - 할인율)
```

예를 들어 할인율이 0.1이라면 10% 할인이라는 뜻입니다. 그래서 `1 - 할인율`을 곱하면 할인 후 실제로 받은 금액이 됩니다. 물건의 정가를 먼저 보고, 할인된 비율만큼 깎아서 최종 금액을 계산하는 방식입니다.

L2 셀을 20,001행까지 채웁니다. 주문이 20,000건이므로, 모든 주문에 같은 매출 계산식을 적용해야 합니다.

확인할 값:

- L2: `22900`
- L3: `35910`
- L4: `105800`

위 값이 맞으면 단가, 수량, 할인율이 제대로 연결되어 계산된 것입니다.

### 7단계. 지역별 매출 집계표 만들기

이 단계에서는 주문별로 계산한 매출을 지역 기준으로 모아 봅니다. 한 줄 한 줄의 주문을 보는 것이 아니라, 지역별로 주문이 몇 건이고 매출이 얼마인지 요약하는 작업입니다.

1. 새 시트를 만듭니다.
2. 시트 이름을 `region_summary`로 바꿉니다.
3. A1:D1에 제목을 입력합니다.

| 셀 | 제목 |
|---|---|
| A1 | `region` |
| B1 | `order_count` |
| C1 | `total_sales` |
| D1 | `avg_order_value` |

4. `orders` 시트의 J2:J20001을 복사해 `region_summary`의 A2에 붙여넣습니다.
5. A열을 선택하고 데이터 탭에서 중복 제거를 실행합니다.
6. A열을 오름차순으로 정렬합니다.

4번부터 6번까지는 지역 목록을 하나씩만 남기는 과정입니다. 주문은 20,000건이지만, 지역 이름은 여러 번 반복됩니다. 집계표에서는 같은 지역이 한 번만 나오면 됩니다.

B2 셀에 입력합니다.

```excel
=COUNTIF(orders!$J$2:$J$20001,A2)
```

`COUNTIF`는 조건에 맞는 행의 개수를 셉니다. 여기서는 A2에 적힌 지역과 같은 주문이 몇 건인지 세는 공식입니다.

C2 셀에 입력합니다.

```excel
=SUMIF(orders!$J$2:$J$20001,A2,orders!$L$2:$L$20001)
```

`SUMIF`는 조건에 맞는 행의 값을 더합니다. 여기서는 A2에 적힌 지역의 주문만 골라서 L열의 매출을 합칩니다.

D2 셀에 입력합니다.

```excel
=C2/B2
```

객단가는 주문 한 건당 평균 매출입니다. 키를 잴 때 전체 키의 합을 사람 수로 나누듯이, 매출 합계를 주문 건수로 나누면 주문 한 건의 평균 금액이 나옵니다.

B2:D2를 마지막 지역 행까지 채웁니다.

검산 기준:

- `order_count` 합계: `20000`
- `total_sales` 합계: `1938489803`
- 서울특별시 매출: `570482124`
- 경기도 매출: `530726511`

검산 값이 맞으면 지역별 집계가 정상적으로 된 것입니다. 특히 `order_count` 합계가 **20000**이어야 합니다. 주문이 빠지거나 중복으로 들어가면 이 숫자가 달라집니다.

## 직접 해보기

### 미션. 등급별 객단가 구하기

이번에는 고객 등급별로 주문 건수, 매출 합계, 객단가를 구해 보세요. 앞에서 지역별로 했던 작업을 등급 기준으로 바꾸는 연습입니다.

고객 등급별로 주문 건수, 매출 합계, 객단가를 구합니다.

1. 새 시트를 만듭니다.
2. 시트 이름을 `grade_summary`로 바꿉니다.
3. A1:D1에 제목을 입력합니다.

| 셀 | 제목 |
|---|---|
| A1 | `grade` |
| B1 | `order_count` |
| C1 | `total_sales` |
| D1 | `avg_order_value` |

4. A2:A5에 등급을 입력합니다.

```text
VIP
골드
실버
브론즈
```

5. B2:D2에 알맞은 `COUNTIF`, `SUMIF`, 나눗셈 공식을 작성합니다.
6. B2:D2를 5행까지 채웁니다.
7. 어떤 등급의 객단가가 가장 높은지 한 문장으로 적습니다.

1번과 2번은 등급별 결과만 따로 보기 위한 공간을 만드는 과정입니다. 3번과 4번은 집계표의 뼈대를 만드는 과정입니다. 5번에서는 주문 표의 등급과 매출을 이용해 계산하고, 6번에서는 같은 공식을 다른 등급에도 적용합니다. 7번에서는 계산 결과를 숫자로만 두지 말고, 사람이 읽을 수 있는 문장으로 해석해 봅니다.

힌트:

- 등급은 `orders` 시트의 K열에 있습니다.
- 매출은 `orders` 시트의 L열에 있습니다.
- 객단가는 매출 합계를 주문 건수로 나눈 값입니다.

지역별 집계에서 지역 이름을 조건으로 썼다면, 여기서는 등급 이름을 조건으로 쓰면 됩니다. 조건이 바뀌어도 `COUNTIF`, `SUMIF`, 나눗셈의 생각 방식은 같습니다.

## 제출 전 확인

아래 값과 맞으면 실습이 정상적으로 완료된 것입니다.

| 항목 | 기준값 |
|---|---:|
| 전체 주문 건수 | 20,000 |
| 전체 매출 | 1,938,489,803 |
| VIP 객단가 | 104,719.97 |
| 골드 객단가 | 100,777.65 |
| 실버 객단가 | 94,709.48 |
| 브론즈 객단가 | 88,787.10 |

가장 높은 객단가는 VIP 등급입니다. 이 문장은 단순한 감상이 아니라, 등급별 매출 합계를 등급별 주문 건수로 나눈 결과를 비교해서 나온 해석입니다.
