## 따라하기 4. pandas로 파일 열기 확인

Colab에서 실습하려면 왼쪽 파일 아이콘을 누르고 `cafe_sales_clean.csv`를 업로드한다. 로컬 PC에서 실행하는 경우에는 수업 루트 폴더에서 실행한다.

아래 코드를 실행한다.

```python
from pathlib import Path
import pandas as pd

path = Path("/mnt/d/ai-data-analysis-course/02_datasets/cafe_sales/cafe_sales_clean.csv")
if not path.exists():
    path = Path("cafe_sales_clean.csv")

df = pd.read_csv(path)

print("행과 열:", df.shape)
print("열 이름:", list(df.columns))
print(df.head())
```

수업용 데이터 기준으로 `행과 열: (10424, 12)`가 나오면 정상이다. Colab에 직접 업로드한 경우에도 파일명만 맞으면 같은 결과가 나온다.
