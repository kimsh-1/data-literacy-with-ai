# S02 핸드아웃. 좋은 데이터의 모양

## 오늘 사용할 파일

아래 두 파일을 사용한다.

- `/mnt/d/ai-data-analysis-course/02_datasets/cafe_sales/cafe_sales_clean.xlsx`
- `/mnt/d/ai-data-analysis-course/02_datasets/cafe_sales/cafe_sales_dirty.xlsx`

두 파일 모두 `sales` 시트가 있으며 컬럼은 12개다. clean 파일은 10,424행, dirty 파일은 10,439행이다. 엑셀에서는 머리글 행까지 포함되므로 화면상 마지막 행 번호는 각각 10,425와 10,440으로 보일 수 있다.

## 따라하기 1. 분석하기 좋은 표 모양 확인

1. `cafe_sales_clean.xlsx`를 연다.
2. `sales` 시트를 확인한다.
3. 1행의 머리글을 왼쪽부터 읽는다.
   - `order_id`, `date`, `weekday`, `hour`, `store`, `category`, `item`, `unit_price`, `qty`, `amount`, `payment`, `member`
4. 2행을 가로로 읽는다.
   - 한 행이 주문 한 건을 설명하는지 확인한다.
5. 각 열을 세로로 훑어본다.
   - `date`에는 날짜만 있다.
   - `store`에는 지점명만 있다.
   - `qty`에는 수량만 있다.
   - `amount`에는 주문금액만 있다.

정리: 이 표는 “주문 한 건이 한 행, 주문의 속성이 각 열”인 구조다. 이런 구조를 피벗테이블과 분석 도구가 반긴다.

## 따라하기 2. dirty 표에서 문제 찾기

1. `cafe_sales_dirty.xlsx`를 연다.
2. `sales` 시트에서 1행 아무 셀이나 클릭한다.
3. 엑셀 상단 메뉴에서 `데이터`를 누른다.
4. `필터`를 클릭한다.
5. 각 컬럼의 필터 버튼을 눌러 값 목록을 확인한다.

### 2-1. 지점 표기 문제 찾기

1. `store` 컬럼의 필터 버튼을 누른다.
2. 값 목록에서 다음 값이 함께 있는지 확인한다.
   - `강남점`
   - `강남 점`
   - `홍대점`
3. `강남점`과 `강남 점`은 같은 지점으로 보아야 한다.

기록할 내용: `store`는 범주형 컬럼이지만 같은 범주의 표기가 섞여 있다.

### 2-2. 상품명 표기 문제 찾기

1. `item` 컬럼의 필터 버튼을 누른다.
2. 검색창에 `아메`를 입력한다.
3. 다음처럼 비슷하지만 다른 값이 있는지 확인한다.
   - `아메리카노`
   - `아메리카`
   - `아메 리카노`
4. 필터 검색창에 `Americano`도 입력해 본다.

기록할 내용: 같은 상품이 여러 이름으로 기록되면 상품별 매출이 나뉘어 계산된다.

### 2-3. 카테고리 앞뒤 공백 찾기

1. `category` 컬럼의 필터 버튼을 누른다.
2. 값 목록에서 `커피`와 ` 커피 `처럼 앞뒤 공백이 있는 값이 따로 보이는지 확인한다.
3. `음료`, `디저트`도 같은 방식으로 확인한다.

기록할 내용: 사람 눈에는 거의 같아도 컴퓨터는 공백이 있으면 다른 값으로 처리할 수 있다.

### 2-4. 단가 컬럼의 숫자와 문자 혼합 찾기

1. `unit_price` 컬럼의 필터 버튼을 누른다.
2. 검색창에 `원`을 입력한다.
3. `4,500원`, `5,500원`처럼 숫자와 문자가 함께 들어간 값을 확인한다.

기록할 내용: 단가 열은 계산에 쓰는 수치형이어야 하므로 `원`과 쉼표를 제거하고 숫자로 맞춰야 한다.

### 2-5. 날짜 형식 혼합 찾기

1. `date` 컬럼의 값을 훑어본다.
2. 다음처럼 날짜 형식이 섞여 있는지 확인한다.
   - `2025-03-23`
   - `2025/03/23`
   - `03-23-2025`

기록할 내용: 날짜가 여러 형식으로 섞이면 월별 추세나 요일 분석 전에 날짜형으로 통일해야 한다.

### 2-6. 결측값 찾기

1. `payment` 컬럼의 필터 버튼을 누른다.
2. 값 목록에서 `(빈 셀)` 또는 비어 있는 항목을 선택한다.
3. 결제수단이 비어 있는 행을 확인한다.
4. 필터를 해제한다.
5. `member` 컬럼에서도 같은 방식으로 빈 셀을 확인한다.

기록할 내용: 비어 있는 값은 무시할지, 별도 범주로 둘지, 원자료를 확인해 채울지 결정해야 한다.

### 2-7. 수량 이상치 찾기

1. `qty` 컬럼의 필터 버튼을 누른다.
2. 값 목록에서 `999`가 있는지 확인한다.
3. `999`만 선택해 해당 행을 본다.

기록할 내용: 카페 주문 수량이 999인 값은 입력 오류일 가능성이 높다. 평균과 총합을 왜곡할 수 있다.

### 2-8. 음수 금액 찾기

1. `amount` 컬럼의 필터 버튼을 누른다.
2. 숫자 필터에서 `보다 작음`을 선택한다.
3. 기준값에 `0`을 입력한다.
4. 음수 금액 행이 있는지 확인한다.

기록할 내용: 환불 데이터라는 별도 설명이 없다면 매출 금액이 음수인 행은 확인이 필요하다.
