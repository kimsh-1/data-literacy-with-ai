## 따라하기 3: 리스트로 매출 여러 개 담기

새 코드 셀에 아래 코드를 입력한다.

```python
sales_list = [120000, 98000, 135000, 170000, 155000]

print(sales_list)
print(len(sales_list))
```

`sales_list`는 월요일부터 금요일까지 매출 5개를 담은 리스트라고 보면 된다. `len()`은 값이 몇 개인지 세는 함수다.

## 따라하기 4: 반복문으로 매출 합계 구하기

이번에는 `sum()` 함수 대신 반복문으로 하나씩 더해 본다. 새 코드 셀에 아래 코드를 입력한다.

```python
total_sales = 0

for amount in sales_list:
    total_sales = total_sales + amount

print(total_sales)
```

기대 결과:

```text
678000
```

코드를 읽는 순서는 다음과 같다.

1. `total_sales = 0`으로 합계를 담을 변수를 만든다.
2. `sales_list`에서 값을 하나씩 꺼내 `amount`라고 부른다.
3. 꺼낸 값을 `total_sales`에 계속 더한다.
4. 반복이 끝나면 `total_sales`에는 전체 합계가 남는다.

주의할 점:

- `for` 줄 끝에는 콜론 `:`을 붙인다.
- 반복할 코드는 반드시 들여쓰기한다.
- 들여쓰기는 보통 스페이스 4칸이다.

## 따라하기 8: 리스트와 딕셔너리로 작은 표 만들기

새 코드 셀에 아래 코드를 입력한다.

```python
daily_sales = [
    {"요일": "월", "매출": 120000},
    {"요일": "화", "매출": 98000},
    {"요일": "수", "매출": 135000},
    {"요일": "목", "매출": 170000},
    {"요일": "금", "매출": 155000},
    {"요일": "토", "매출": 210000},
    {"요일": "일", "매출": 190000},
]

for row in daily_sales:
    print(row["요일"], row["매출"])
```

요일과 매출이 한 줄씩 출력된다. 여기서 `row`는 반복문이 지금 보고 있는 한 행이다.

## 직접 해보기: 평일 데이터만 필터링하기

아래 조건을 만족하는 코드를 작성한다.

목표:

1. `daily_sales`에서 평일인 `월`, `화`, `수`, `목`, `금` 행만 고른다.
2. 고른 행을 `weekday_rows` 리스트에 담는다.
3. 평일 매출 합계와 평균을 계산한다.
4. 평일 행수, 평일 합계, 평일 평균을 출력한다.

힌트 코드:

```python
weekday_names = ["월", "화", "수", "목", "금"]
weekday_rows = []
weekday_total = 0

for row in daily_sales:
    if row["요일"] in weekday_names:
        weekday_rows.append(row)
        weekday_total = weekday_total + row["매출"]

weekday_average = weekday_total / len(weekday_rows)

print(weekday_rows)
print(weekday_total)
print(weekday_average)
```

기대 결과:

```text
평일 행수: 5
평일 매출 합계: 678000
평일 매출 평균: 135600.0
```

## 오류가 났을 때 먼저 볼 것

오류가 나면 빨간 메시지의 맨 아래 줄부터 읽는다.

자주 만나는 오류:

| 오류 | 흔한 원인 | 확인할 것 |
|---|---|---|
| `NameError` | 변수를 만들기 전에 사용함 | 위 셀을 실행했는가 |
| `SyntaxError` | 문법이 틀림 | 콜론, 따옴표, 괄호가 빠졌는가 |
| `IndentationError` | 들여쓰기가 틀림 | `if`, `for` 아래 줄이 들여쓰기 되었는가 |
| `TypeError` | 자료형이 맞지 않음 | 숫자와 글자를 섞어 계산하지 않았는가 |

오류는 실패가 아니라 파이썬이 어느 줄을 다시 보라고 짚어 주는 신호다.
