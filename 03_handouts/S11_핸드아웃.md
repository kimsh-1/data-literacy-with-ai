## 따라하기 2: DataFrame과 Series 확인하기

DataFrame은 표 전체이고, Series는 열 하나입니다. 엑셀로 보면 DataFrame은 시트 한 장, Series는 그 안의 금액 열 하나라고 생각하면 됩니다.

이 구분을 먼저 확인해야 나중에 “표 전체를 다루는 코드인지, 열 하나를 다루는 코드인지” 덜 헷갈립니다.

```python
print(type(df))

amount_series = df["amount"]
print(type(amount_series))
print(len(amount_series))
```

확인할 점:

- `df`는 `DataFrame`입니다.
- `df["amount"]`는 `Series`입니다.
- `amount_series`의 길이는 전체 행수와 같은 10,424입니다.

엑셀로 비유하면 `df`는 시트 전체이고, `amount_series`는 금액 열 하나입니다. 표 전체와 열 하나를 구분하는 감각이 pandas의 출발점입니다.

## 따라하기 7: 강남점만 필터링하기

### 1단계. 조건 만들기

필터링을 하려면 먼저 어떤 행을 남길지 조건을 만들어야 합니다. 엑셀 필터에서 `강남점`만 체크하기 전에, pandas에서는 각 행이 강남점인지 아닌지를 True와 False로 표시합니다.

```python
gangnam_mask = df["store"] == "강남점"

print(gangnam_mask.head(10))
```

`gangnam_mask`에는 True와 False가 들어 있습니다. 각 행의 지점이 `강남점`이면 True, 아니면 False입니다.

이 True/False 열을 불리언 마스크라고 부릅니다. “조건에 맞는 행에는 표시를 해 둔 체크 목록”이라고 생각하면 됩니다.

### 2단계. True인 행만 남기기

이제 방금 만든 체크 목록을 사용해서 True인 행만 남깁니다. 엑셀에서 필터를 적용하면 조건에 맞는 행만 보이는 것과 같은 과정입니다.

```python
gangnam = df.loc[gangnam_mask]

print(gangnam.shape)
print(gangnam.head())
```

확인할 값:

```text
(6196, 12)
```

엑셀에서 지점 필터를 열고 `강남점`만 체크한 결과와 같습니다. 6,196개의 행이 남고, 열은 원래처럼 12개입니다.

## 따라하기 8: 강남점 주문을 금액 큰 순서로 정렬하기

강남점 주문만 남겼다면, 이제 금액이 큰 주문부터 보겠습니다. 정렬은 큰 값이나 작은 값을 먼저 보고 싶을 때 쓰는 기능입니다.

```python
display_cols = ["order_id", "date", "store", "category", "item", "qty", "amount"]

gangnam_sorted = gangnam.sort_values(
    ["amount", "date", "order_id"],
    ascending=[False, True, True],
)

print(gangnam_sorted.loc[:, display_cols].head(10))
```

해석:

- `amount`는 큰 금액부터 봅니다.
- 같은 금액이면 `date`, `order_id` 순서로 보기 좋게 정렬합니다.
- `display_cols`는 화면에 보여줄 열만 고른 것입니다.

`sort_values()`는 엑셀에서 특정 열을 기준으로 오름차순이나 내림차순 정렬하는 기능과 같습니다. 여기서는 금액을 가장 먼저 보고 싶기 때문에 `amount`를 정렬 기준으로 둡니다.

## 따라하기 9: loc와 iloc 비교하기

`.loc`는 이름 기준, `.iloc`는 위치 기준입니다. 사전에서 단어 이름으로 찾는 방식이 `.loc`에 가깝고, 책장에서 몇 번째 칸인지 세어서 찾는 방식이 `.iloc`에 가깝습니다.

```python
print(df.loc[0, "store"])
print(df.iloc[0, 4])
```

둘 다 아래 값을 출력합니다.

```text
홍대점
```

하지만 뜻은 다릅니다.

- `df.loc[0, "store"]`: 인덱스 라벨 0번 행의 `store` 열
- `df.iloc[0, 4]`: 0번째 행, 4번째 열

여러 행과 여러 열을 고를 수도 있습니다.

```python
print(df.loc[0:2, ["date", "store", "amount"]])
print(df.iloc[0:3, [1, 4, 9]])
```

주의할 점:

- `.loc[0:2]`는 라벨 기준이라 0, 1, 2가 포함됩니다.
- `.iloc[0:3]`는 위치 기준이라 0, 1, 2번째 행까지만 포함되고 3번째는 제외됩니다.

처음에는 열 이름이 직접 보이는 `.loc`가 더 읽기 쉽습니다. 다만 위치 번호로 골라야 할 때는 `.iloc`를 씁니다.

## 직접 해보기: 조건 행 추출

아래 조건을 모두 만족하는 주문만 추출해 보세요. 지금까지 배운 불리언 마스크, `.loc`, 정렬을 한 번에 연결하는 연습입니다.

할 일:

1. 조건을 `mission_mask`라는 이름으로 만듭니다. 조건을 이름으로 저장해 두면, 어떤 행을 남길지 다시 확인하기 쉽습니다.
2. `df.loc[mission_mask]`로 조건에 맞는 행만 추출합니다. True인 행만 남기는 필터 결과를 만드는 단계입니다.
