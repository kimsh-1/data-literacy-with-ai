## 따라하기 9: loc와 iloc 비교하기

`.loc`는 이름 기준, `.iloc`는 위치 기준이다.

```python
print(df.loc[0, "store"])
print(df.iloc[0, 4])
```

둘 다 아래 값을 출력한다.

```text
홍대점
```

하지만 뜻은 다르다.

- `df.loc[0, "store"]`: 인덱스 라벨 0번 행의 `store` 열
- `df.iloc[0, 4]`: 0번째 행, 4번째 열

여러 행과 여러 열을 고를 수도 있다.

```python
print(df.loc[0:2, ["date", "store", "amount"]])
print(df.iloc[0:3, [1, 4, 9]])
```

주의할 점:

- `.loc[0:2]`는 라벨 기준이라 0, 1, 2가 포함된다.
- `.iloc[0:3]`는 위치 기준이라 0, 1, 2번째 행까지만 포함되고 3번째는 제외된다.

## 직접 해보기: 조건 행 추출

아래 조건을 모두 만족하는 주문만 추출한다.

조건:

1. 지점이 `강남점`이다.
2. 카테고리가 `커피`이다.
3. 주문 금액 `amount`가 10,000원 이상이다.

할 일:

1. 조건을 `mission_mask`라는 이름으로 만든다.
2. `df.loc[mission_mask]`로 조건에 맞는 행만 추출한다.
3. `amount`가 큰 순서로 정렬한다.
4. 조건에 맞는 행수, 총금액, 평균금액을 출력한다.
5. `order_id`, `date`, `store`, `category`, `item`, `qty`, `amount` 열만 골라 상위 10행을 출력한다.

힌트:

```python
mission_mask = (
    (df["store"] == "강남점")
    & (df["category"] == "커피")
    & (df["amount"] >= 10000)
)
```

빈칸을 채워 완성한다.

```python
mission = df.loc[mission_mask, display_cols].sort_values(
    ["amount", "date", "order_id"],
    ascending=[False, True, True],
)

print("행수:", ________)
print("총금액:", ________)
print("평균금액:", ________)
print(________)
```

완료 후 스스로 확인할 것:

- 조건 3개를 모두 괄호로 감쌌는가
- 조건 사이에 `&`를 썼는가
- `and`를 쓰지 않았는가
- 정렬 결과가 금액 큰 순서로 보이는가
- 행수, 총금액, 평균금액을 숫자로 출력했는가
